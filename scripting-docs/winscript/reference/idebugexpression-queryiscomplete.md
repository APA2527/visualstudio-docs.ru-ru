# <a name="idebugexpressionqueryiscomplete"></a>IDebugExpression::QueryIsComplete
Определяет, завершена ли операция.  
  
## <a name="syntax"></a>Синтаксис  
  
```cpp
HRESULT QueryIsComplete();  
```  
  
#### <a name="parameters"></a>Параметры  
 Этот метод не принимает параметров.  
  
## <a name="return-value"></a>Возвращаемое значение  
 Метод возвращает `HRESULT`. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.  
  
|значения|Описание|  
|-----------|-----------------|  
|`S_OK`|Метод выполнен, и операция завершена.|  
|`S_FALSE`|Операция все еще находится в состоянии ожидания.|  
  
## <a name="remarks"></a>Заметки  
 Этот метод определяет, завершена ли операция.
