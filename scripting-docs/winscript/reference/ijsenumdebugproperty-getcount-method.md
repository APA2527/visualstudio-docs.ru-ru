# <a name="ijsenumdebugpropertygetcount-method"></a>Метод IJsEnumDebugProperty::GetCount

## <a name="syntax"></a>Синтаксис  
  
```cpp
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `pCount`  
 заполняет Число свойств.
