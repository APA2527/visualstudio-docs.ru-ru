# <a name="iactivescriptgarbagecollector-interface"></a>Интерфейс IActiveScriptGarbageCollector
Предоставляет метод запуска сборки мусора. Этот интерфейс должен быть реализован механизмами активных скриптов, которые хотят очищать свои ресурсы.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|Запускает сборку мусора.|
