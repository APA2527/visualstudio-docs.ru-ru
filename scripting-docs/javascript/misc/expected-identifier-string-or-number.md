# <a name="expected-identifier-string-or-number"></a>Ожидался идентификатор, строка или число
Для объявления объектного литерала использовался неверный синтаксис литерала. Свойства объектного литерала должны быть идентификатором, строкой или числом. Объектный литерал (также называемый "инициализатором объекта") состоит из разделенного запятыми списка пар "свойство: значение", заключенных в квадратные скобки. Пример:  
  
```JavaScript  
var point = {x:1.2, y:-3.4};  
```  
  
### <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Убедитесь, что используется правильный синтаксис Literal.
