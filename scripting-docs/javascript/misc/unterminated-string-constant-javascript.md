# <a name="unterminated-string-constant-javascript"></a>Строковая константа без признака завершения (JavaScript)
Не удалось завершить вашей строковая константа с закрывающей кавычкой. Строковые константы должны заключаться заключенный в кавычки.  
  
> [!NOTE]
> Можно использовать пары одинарные или двойные кавычки. Двойные кавычки могут содержаться в строках, заключенных в одинарные кавычки, а одинарные кавычки могут содержаться в строках, заключенных в двойные кавычки.  
  
### <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Добавьте знак закрывающей кавычки в конец строки.
