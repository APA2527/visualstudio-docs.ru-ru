# <a name="array-or-arguments-object-expected"></a>Ожидается объект массива или аргументов
Вы не указали массив в качестве аргумента. Эта ошибка относится только к методу **Function. prototype. Apply** . Если параметр указан, то второй аргумент этой функции должен быть либо объектом `Array`, либо объектом **arguments** .  
  
### <a name="to-correct-this-error"></a>Исправление ошибки  
  
- В качестве второго аргумента укажите либо `Array`, либо объект **arguments** .
