# <a name="invalid-character-javascript"></a>Недопустимый символ (JavaScript)
Вы состояли идентификатор, используя символ (или символы), который не распознается компилятором [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] как допустимый. Допустимые символы используют следующие правила.  
  
- Первый символ должен быть буквой ASCII (как в верхнем, так и в нижнем регистре) или символом подчеркивания (_).  
  
- Последующие символы могут быть буквами ASCII, цифрами или символами подчеркивания.  
  
- Имя идентификатора не может быть зарезервированным словом.  
  
### <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Старайтесь не использовать символы, которые не являются частью определения языка [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].
