# <a name="illegal-assignment-javascript"></a>Недопустимое присваивание (JavaScript)
Предпринята попытка присвоить значение идентификатора только для чтения. Невозможно присвоить значение для идентификатора только для чтения. Например узел определенные объекты и внешнего COM-объектов.  
  
### <a name="to-correct-this-error"></a>Исправление ошибки  
  
- Не присваивайте значения идентификаторов только для чтения.
