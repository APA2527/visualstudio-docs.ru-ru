# <a name="enterprise-mobility--security-ems-in-visual-studio-subscriptions"></a>Enterprise Mobility + Security (EMS) в подписках Visual Studio

Преимущество Enterprise Mobility + Security, которое ранее было доступно отдельно в выбранных подписках Visual Studio, теперь включено в подписку Microsoft 365 для разработчиков, входящую в те же выбранные подписки Visual Studio.  

В результате преимущество Enterprise Mobility + Security больше не будет предлагаться отдельно.  

Чтобы активировать подписку Microsoft 365 для разработчиков, выполните следующие действия:
1. Войдите в [портал подписчиков Visual Studio](https://my.visualstudio.com/benefits?wt.mc_id=o~msft~docs).
1. Найдите элемент "Подписка Microsoft 365 для разработчиков" и щелкните **Активировать**.  
1. Следуйте инструкциям на экране, чтобы завершить создание учетной записи.  

Ознакомьтесь с дополнительными сведениями об активации [подписки Microsoft 365 для разработчиков](vs-m365.md).
