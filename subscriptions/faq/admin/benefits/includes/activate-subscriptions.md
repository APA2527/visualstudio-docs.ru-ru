## <a name="how-do-my-end-users-activate-their-visual-studio-subscription"></a>Как мои конечные пользователи могут активировать свою подписку Visual Studio?

Подписка активируется и становится доступна для использования в момент ее назначения на [портале администрирования Visual Studio](https://manage.visualstudio.com/subscribers). После этого ваши подписчики могут выполнить вход на [портал Visual Studio для подписчиков](https://my.visualstudio.com/) или в интегрированную среду разработки Visual Studio и получат сообщение электронной почты с уведомлением о назначении подписки.
