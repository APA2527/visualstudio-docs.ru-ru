## <a name="how-do-my-developers-activate-software-using-product-keys-from-their-visual-studio-subscription"></a>Как моим разработчикам активировать программное обеспечение с помощью ключей продукта из своей подписки Visual Studio?

В зависимости от того, какой продукт был скачан, ваш подписчик может активировать его, выполнив вход с использованием привязанного к подписке адреса электронной почты, либо [запросив ключ продукта](../../../../product-keys.md) на портале Visual Studio для подписчиков.
