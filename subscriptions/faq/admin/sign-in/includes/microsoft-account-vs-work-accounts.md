## <a name="what-is-the-difference-between-using-a-microsoft-account-vs-work-account"></a>В чем разница между использованием учетной записи Майкрософт и рабочей учетной записи?

Учетная запись Майкрософт (MSA) — это личная учетная запись электронной почты, которая используется для доступа к службам Майкрософт в тех случаях, когда организация не предоставляет управляемый клиент для Microsoft 365 или Azure Active Directory. Рабочая учетная запись принадлежит организации, которая выдала ее, и предоставляет доступ к службам в клиентах Microsoft 365 или Azure Active Directory. См. видео с [подробной сравнительной характеристикой MSA (учетных записей Майкрософт) и рабочих учетных записей (Azure Active Directory или Microsoft 365), а также сведениями о влиянии их использования на работу организации](https://aka.ms/MSAvsAAD).
