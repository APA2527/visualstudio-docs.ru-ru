## <a name="what-software-downloads-are-available"></a>Какое программное обеспечение доступно для скачивания?

Список программного обеспечения, доступного для скачивания, зависит от типа используемой подписки. Если у вас несколько подписок, в каждой из них будут видны скачиваемые файлы, доступные в подписке самого высокого уровня. Благодаря этому вы сможете легко скачать нужные файлы, не переключаясь между подписками.

Ознакомьтесь с [перечнем доступных для скачивания файлов для каждого уровня подписки](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx). Дополнительные сведения о скачиваемых файлах см. в следующих статьях:

- [Доступность продуктов](../../../../product-availability.md)
- [Скачивание программного обеспечения](../../../../download-software.md)
