## <a name="what-are-the-licensing-terms-for-visual-studio-subscriptions"></a>Каковы условия лицензирования для подписок Visual Studio? 

Все подписки Visual Studio лицензируются по принципу "на пользователя".  Каждый лицензированный пользователь может устанавливать и использовать программное обеспечение на любом числе устройств в целях проектирования, разработки, тестирования и демонстрации своих программ.  Подписки Visual Studio также позволяют лицензированному пользователю использовать пробные версии ПО и симулировать среды клиентов в целях диагностирования проблем, связанных с их программами.  Таким образом, у каждого дополнительного лица, использующего программное обеспечение, также должна быть лицензия. 

Дополнительные сведения представлены в [документе о лицензировании Microsoft Visual Studio](https://aka.ms/vslicensing).
