## <a name="my-organization-has-multiple-visual-studio-subscribers-can-their-monthly-azure-credits-be-pooled"></a>В моей организации несколько подписчиков Visual Studio. Можно ли объединить их кредиты Azure?

Объединение кредитов невозможно. Это преимущество предоставляется подписчику Visual Studio исключительно для личного пользования.
