## <a name="im-unable-to-download-or-activate-visual-studio-code"></a>Не удается скачать или активировать Visual Studio Code

Visual Studio Code — это бесплатный редактор кода, и для его использования не требуется платная подписка Visual Studio. Вы можете [загрузить его непосредственно](https://code.visualstudio.com/download) с веб-сайта Visual Studio.
