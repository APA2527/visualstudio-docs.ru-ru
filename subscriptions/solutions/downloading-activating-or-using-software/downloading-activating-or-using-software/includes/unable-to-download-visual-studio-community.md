## <a name="im-unable-to-download-or-activate-visual-studio-community"></a>Не удается скачать или активировать Visual Studio Community
Visual Studio Community — это бесплатная упрощенная версия интегрированной среды разработки, и для ее использования не требуется платная подписка Visual Studio. Вы можете [загрузить ее непосредственно](https://visualstudio.microsoft.com/vs/community/) с веб-сайта Visual Studio.
