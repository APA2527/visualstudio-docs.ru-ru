## <a name="im-unable-to-download-or-activate-visual-studio-for-mac"></a>Мне не удается скачать или активировать Visual Studio для Mac

Как подписчик Visual Studio вы получаете доступ к интегрированной среде разработки Visual Studio. Visual Studio для Mac — это отдельный файл для загрузки, который можно найти на [странице загрузки](https://my.visualstudio.com/Downloads) на портале подписок.  

## <a name="find-and-download-visual-studio-for-mac"></a>Поиск и скачивание Visual Studio для Mac 
1. Войдите на [портал подписок](https://my.visualstudio.com/benefits) и откройте [отсортированное представление](https://my.visualstudio.com/Downloads?q=Visual%20Studio%20for%20mac&pgroup=) с доступными загрузками Visual Studio для Mac. Мы предлагаем только актуальные версии продуктов. 
2. Следуйте инструкциям по скачиванию и настройке. 
1. Используя адрес электронной почты, которому назначена подписка, вы сможете войти в интегрированную среду разработки сразу после установки. Ключ продукта не требуется.

## <a name="no-download-available"></a>Нет файлов, доступных для скачивания 
Список программного обеспечения, доступного для скачивания, зависит от типа используемой подписки. Если вы не видите нужный файл для загрузки, возможно, он недоступен для вас. Чтобы проверить доступные подписки, войдите на [страницу подписок](https://my.visualstudio.com/subscriptions) портала. Можно [загрузить полный список](https://download.microsoft.com/download/1/5/4/15454442-CF17-47B9-A65D-DF84EF88511B/Visual_Studio_by_Subscription_Level.xlsx)  доступных программ для всех типов подписок.
