# <a name="properties-of-connectors"></a>Свойства соединителей
Соединители представляют собой доменная связь в созданном конструкторе.

 Дополнительные сведения см. в разделе [способ определения доменного языка](../modeling/how-to-define-a-domain-specific-language.md). Дополнительные сведения об использовании этих свойств см. в разделе [Настройка и расширение доменного языка](../modeling/customizing-and-extending-a-domain-specific-language.md).

 Соединители имеют свойства, которые перечислены в следующей таблице.

|Свойство|Описание|Значение по умолчанию|
|-|-|-|
|Цвет|Цвет этого соединителя.|Черный|
|Тип штриха|Тип штриха для линии данного соединителя (сплошная, тире, точка, DashDot, DashDotDot или настраиваемый).|Сплошная|
|Стиль на стороне источника|Стиль окончания на источнике для этого соединителя (HollowArrow, EmptyArrow, FilledArrow, EmptyDiamond, FilledDiamond или нет).|Нет|
|Стиль на стороне цели|Стиль окончания на целевой объект для этого соединителя (HollowArrow, EmptyArrow, FilledArrow, EmptyDiamond, FilledDiamond или нет).|Нет|
|Цвет текста|Цвет, используемый для декораторов текста, которые связаны с этим соединителем.|Черный|
|Thickness|Толщина линии для данного соединителя, измеренная в дюймах.|0.03125|
|Модификатор доступа|Уровень доступа класса (`public` или `internal`).|Public|
|Настраиваемые атрибуты|Используется для добавления атрибутов для класса исходного кода, созданный из этого соединителя.|\<none>|
|Создает двойной производным|Если `True`, будет создан базовый класс и разделяемый класс (для поддержки настройки посредством переопределений). Дополнительные сведения см. в разделе [переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|
|Имеет пользовательский конструктор|Если `True`, пользовательский конструктор будет предоставляться в исходном коде. Дополнительные сведения см. в разделе [переопределение и расширение созданных классов](../modeling/overriding-and-extending-the-generated-classes.md).|False|
|Модификатор наследования|Описывает тип наследования класса исходного кода, созданный из соединителя (`none`, `abstract` или `sealed`).|Нет|
|Базовый соединитель|Базовый класс этого соединителя.|(нет)|
|name|Имя этого соединителя.|Текущее имя|
|Пространство имен|Пространство имен, которое входит в состав этого соединителя.|Текущее пространство имен|
|Тип подсказки|Как подсказка определяется (фиксированное значение, переменная или none). Если так, то значение `Fixed Tooltip Text` свойство используется в качестве подсказки; Если переменная, то подсказки определен в пользовательском коде.|\<none>|
|Примечания|Неофициальные примечания, связанные с этим соединителем.|\<none>|
|Стиль маршрутизации|Стиль, используемый для маршрутизации соединителя. Объект `Rectilinear` соединитель устанавливает прямоугольные включает при необходимости; `Straight` соединитель — нет.|Ломаные линии|
|Цвет, предоставляемого как свойство<br /><br /> Тип предоставляемого штриха как свойство<br /><br /> Толщина, предоставляемого как свойство<br /><br /> Предоставляет цвет текста|Если `True`, пользователь может задать свойство заявленным фигуры. Для этого щелкните правой кнопкой мыши определения фигуры и нажмите кнопку **добавить предоставленный**.|False|
|Описание|Используется для документирования созданного конструктора.|\<none>|
|Отображаемое имя|Имя, которое будет отображаться в созданном конструкторе для этого соединителя.|\<none>|
|Фиксированный текст всплывающей подсказки|Текст, используемый для фиксированной подсказки.|\<none>|
|Ключевое слово Help|Ключевое слово, используемое для индексации справки F1 для данного элемента.|\<none>|
