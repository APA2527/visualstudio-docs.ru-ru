# <a name="spy-views"></a>Представления Spy++
Spy++ отображает древовидные представления окон, сообщений, процессов и потоков, доступных на вашем компьютере.

## <a name="in-this-section"></a>В этом разделе
 [Представление окон](../debugger/windows-view.md). Отображает представление доступных окон и элементов управления в виде дерева.

 [Представление сообщений](../debugger/messages-view.md). Отображает поток сообщений, связанный с окном, процессом или потоком.

 [Представление процессов](../debugger/processes-view.md). Отображает представление активных процессов в виде дерева.

 [Представление потоков](../debugger/threads-view.md). Отображает представление активных потоков в виде дерева.
