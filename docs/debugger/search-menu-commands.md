# <a name="search-menu-commands"></a>Команды меню "Поиск"
В меню **Поиск** содержатся команды для поиска конкретных окон.

 Команда **Найти окно, процесс, поток, сообщение** открывает диалоговое окно поиска **Окно**/**Процесс**/**Поток**/**Сообщение**, где вы можете использовать **Инструмент поиска** или задать нужные условия поиска.

 **Поиск следующих.** Выполняет поиск следующего соответствующего окна, процесса, потока или сообщения в текущем представлении. Этот пункт меню (и соответствующая кнопка на панели инструментов) доступен только при наличии допустимого результата поиска, который не является уникальным.

 **Поиск предыдущих.** Выполняет поиск предыдущего соответствующего окна, процесса, потока или сообщения в текущем представлении. Этот пункт меню (и соответствующая кнопка на панели инструментов) доступен только при наличии допустимого результата поиска, который не является уникальным.
