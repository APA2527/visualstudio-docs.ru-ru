# <a name="error-unable-to-connect-to-sql-server-on-remote-machine"></a>Ошибка: не удается подключиться к SQL Server на удаленном компьютере
Не удалось подключиться к SQL Server в *имени*удаленного компьютера. Доступ запрещен. Проверьте, что на удаленном компьютере установлен удаленный отладчик. Если удаленный компьютер не входит в домен, или если Visual Studio выполняется с локальной учетной записью, удаленный компьютер должен иметь учетную запись с тем же именем и паролем, что и локальная учетная запись.

### <a name="to-correct-this-error"></a>Исправление ошибки

- См. статью об [удаленной отладке](../debugger/remote-debugging.md).
