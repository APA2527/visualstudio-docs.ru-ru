# <a name="how-to-display-spy-views"></a>Практическое руководство. Отображение представлений Spy++
При запуске Spy + + открывается в [представлении Windows](../debugger/windows-view.md), в виде дерева всех окон и элементов управления, доступных на вашем компьютере. Это корневое окно с названием «Windows 1». Также доступны три других представления Spy + +: [представление сообщений](../debugger/messages-view.md), [представление процессов](../debugger/processes-view.md)и [представление потоков](../debugger/threads-view.md). Вы можете отобразить нужное представление в меню **Spy** . Дополнительные сведения см. в разделе [представления Spy + +](../debugger/spy-increment-views.md).
