# <a name="view-dlls-and-executables-in-the-modules-window-c-c-visual-basic-f"></a>Просмотр библиотек DLL и исполняемых файлов в окно "Модули" (C#, C++, Visual Basic, F#)

Во время отладки Visual Studio, **модули** окне перечисляются и отображаются сведения о DLL и исполняемые файлы (*.exe* файлы) используемую приложением.

> [!NOTE]
> При отладке скрипта SQL и окна "Модули" недоступен.

## <a name="use-the-modules-window"></a>Использование окна модулей

Чтобы открыть окно "Модули" во время отладки, выберите **Отладка** > **Windows** > **модули** (или нажмите клавишу **Ctrl + Alt + U** ).

По умолчанию модули в окне **Модули** упорядочены в порядке загрузки. Чтобы отсортировать по любому столбцу окно, выберите заголовок в верхней части столбца.

## <a name="load-symbols"></a>Загрузить символы

**Состояние символов** столбца в **модули** окно показывает каких модулей загружены символы отладки. Если состояние — **загрузка символов пропущена**, **невозможно найти или открыть PDB-файл**, или **загрузка отключена параметром включения/исключения**, можно загружать символы вручную. Дополнительные сведения о загрузке и использовании символов см. в разделе [Указание файлов символов (.pdb) и исходных файлов](../debugger/specify-symbol-dot-pdb-and-source-files-in-the-visual-studio-debugger.md).

**Загрузка символов вручную**

1. В **модули** окно, щелкните правой кнопкой мыши модуль для символов, которые еще не загружен.

   - Выберите **загрузить данные символов** Дополнительные сведения о том, почему не удалось загрузить символы.

   - Выберите **загрузить символы** загрузка символов вручную.

1. Если не загружать символы, выберите **параметры символов** открыть **параметры** диалоговое окно и указать или изменить символ, идет загрузка местоположений.

   Можно загрузить символы с открытых серверов символов Майкрософт или других серверов или загрузить символы из папки на компьютере. Дополнительные сведения см. в разделе [определение расположения символов и поведения при загрузке](../debugger/specify-symbol-dot-pdb-and-source-files-in-the-visual-studio-debugger.md#BKMK_Specify_symbol_locations_and_loading_behavior).

**Чтобы изменить параметры поведения загрузки символов:**

1. В окне **Модули** щелкните правой кнопкой мыши любой модуль.

1. Выберите **параметры символов**.

1. Выберите **загрузить все символы**, или выберите, какие модули для включения или исключения.

1. Нажмите кнопку **ОК**. Изменения вступят в силу во время следующего сеанса отладки.

**Изменение поведения для конкретного модуля загрузки символов.**

1. В окне **Модули** щелкните правой кнопкой мыши требуемый модуль.

1. В контекстном меню, выберите или отмените выбор **всегда нагрузки автоматически**. Изменения вступят в силу во время следующего сеанса отладки.
