# <a name="messages-menu-commands"></a>Команды меню "Сообщения"
**Сообщений** меню доступны команды для управления представлениями сообщений.

 **Запуск и остановка ведения журнала** начинает или останавливает просмотреть сообщение.

 **Параметры ведения журнала** открывает **параметры сообщения** диалоговое окно, в котором можно указать условия поиска сообщений.

 **Очистить журнал** очищает содержимое активного представления сообщений.
