# <a name="messages-menu-commands"></a>Команды меню "Сообщения"
В меню **Сообщения** содержатся команды для управления представлениями сообщений.

 **Ведение журнала запуска и остановки** — запускает или останавливает представление сообщений.

 **Параметры ведения журнала** — открывает диалоговое окно **Параметры сообщений**, в котором можно указать условия поиска сообщений.

 **Очистить журнал** — удаляет содержимое активного представления сообщений.
