# <a name="view-menu-commands"></a>Команды меню "Вид"
В меню **Вид** содержатся команды для управления пользовательским интерфейсом Spy++.

 **Шрифт**. Позволяет открыть диалоговое окно **Шрифт**, в котором можно выбрать шрифт для использования в текущем представлении Spy++.

 **Панель инструментов**. Позволяет скрыть или отобразить панель инструментов.

 **Строка состояния**. Позволяет скрыть или отобразить строку состояния.

 **Свойства**. Позволяет открыть диалоговое окно **Свойства** для активного представления.
