# <a name="tree-menu-commands"></a>Команды меню "Дерево"
Меню **Дерево** содержит команды для развертывания или свертывания представлений.

 **Развернуть один уровень**. Разворачивает выбранный элемент в системном представлении, отображая один уровень под элементом.

 **Развернуть ветвь**. Разворачивает все уровни под выбранным элементом.

 **Развернуть все**. Разворачивает и отображает все уровни в системном представлении.

 **Свернуть**. Сворачивает все уровни под выбранным элементом.
