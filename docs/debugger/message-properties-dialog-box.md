# <a name="message-properties-dialog-box"></a>Диалоговое окно "Свойства сообщения"
Используйте это диалоговое окно, чтобы получить дополнительные сведения о конкретном сообщении. Чтобы отобразить это диалоговое окно, перемещение фокуса к [представления сообщений](../debugger/messages-view.md) окна. Выберите любой узел сообщения в дереве, а затем выберите **свойства** из **представление** меню.

 **Общие** вкладка — отображается единственная вкладка. Доступны следующие параметры:

 **Дескриптор окна** уникальный идентификатор этого окна. Номера дескрипторов окон используются повторно; они определяют окна только в течение времени существования этого окна. Щелкните это значение, чтобы просмотреть свойства этого окна.

 **Уровень вложенности** глубину вложенности сообщения, где 0 — без вложенности.

 **Сообщение** номер, состояние и имя выбранного сообщения окна.

 **lResult** значение *lResult* параметр, если таковые имеются.

 **wParam** значение *wParam* параметр, если таковые имеются.

 **lParam** значение *lParam* параметр, если таковые имеются. Это значение декодируется в том случае, если он является указателем на строку или структуру.
