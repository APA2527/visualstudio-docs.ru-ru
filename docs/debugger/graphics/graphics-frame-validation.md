# <a name="graphics-frame-validation"></a>Проверка кадров графики
<!-- VERSIONLESS -->
Visual Studio 2017 и более поздней версии поддерживают средство **проверки кадров** .  В окне Проверка кадров отображаются ошибки и предупреждения, связанные со списком событий.  Чтобы открыть это окно, выберите меню **view > проверки кадра** .

![Проверка кадра](media/gfx_diag_frame_validation.png)

Нажмите кнопку **запустить проверку** в левом верхнем углу, чтобы начать анализ.  В зависимости от сложности кадра выполнение может занять несколько минут.  Данные, которые здесь отображаются, являются сочетанием из двух источников: сообщения, которые автоматически порождаются при включении [слоев SDK](/windows/desktop/direct3d11/overviews-direct3d-11-devices-layers) , а также данные, собираемые из собственного внутреннего отслеживания состояния средства. После завершения вы увидите несколько столбцов данных:

| **Столбец** | **Описание** |
|------------| - |
| Идентификатор события | Идентификатор, который сопоставляется с записью в окне " [список событий](graphics-event-list.md) ". |
| Серьезность | Повреждение, ошибка, предупреждение, информация или сообщение. |
| Категория | Определение приложений, разность, инициализация, очистка, компиляция, создание состояния, Настройка состояния, получение состояния, выполнение, обработка ресурсов, шейдер, избыточность и неиспользуемые. |
| Сообщение | Сообщение, связанное с событием. |
| событие | Событие, связанное с ошибкой или предупреждением. |
