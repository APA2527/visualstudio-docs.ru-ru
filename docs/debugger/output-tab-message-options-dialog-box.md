# <a name="output-tab-message-options-dialog-box"></a>Вкладка "Вывод" диалогового окна "Параметры сообщения"
Используйте **вывода** tab, чтобы указать, какие данные из каждого сообщения в список в [представления сообщений](../debugger/messages-view.md). Для отображения [диалоговое окно "Параметры сообщения"](../debugger/message-options-dialog-box.md), выберите **сообщений журнала** из **Spy** меню.

 Следующие параметры доступны на **вывода** вкладке:

 **Номера строк** отображение номеров строк.

 **Уровень вложенности сообщений** префикса вложенных сообщений с помощью одного периода уровень.

 **Параметры необработанные сообщения** отображения шестнадцатеричного **wParam** и **lParam** значения.

 **Декодировать параметры сообщения** отображаются результаты декодирования конкретного сообщения **wParam** и **lParam** значения.

 **Необработанные значения, возвращают** отображения шестнадцатеричного **lResult** возвращаемое значение.

 **Декодировать возвращаемые значения** отображаются результаты декодирования конкретного сообщения **lResult** возвращаемое значение.

 **Сообщения времени начала координат** время, прошедшее с момента запуска системы Windows (только для отправленных сообщений).

 **Положение указателя мыши сообщения** экранные координаты указателя мыши, когда было сгенерировано сообщение (только для отправленных сообщений).

 **Строк максимум** ограничить число строк, которые сохраняются в выбранном представлении сообщений.

 **Сообщения журнала также файл** указав выходной файл журнала сообщений. Этот выходной файл записывается одновременно с окном сообщения журнала.

 **Сохранить параметры по умолчанию** сохранить указанные выше параметры для новых окон потоков сообщений. Эти параметры сохраняются при закрытии окна Spy ++.
