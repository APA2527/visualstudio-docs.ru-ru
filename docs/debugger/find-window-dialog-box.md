# <a name="find-window-dialog-box"></a>Диалоговое окно "Поиск окна"
Это диалоговое окно используется для выбора окна и отображения его свойств или сообщений. Чтобы отобразить это диалоговое окно, выберите **найти окно** из **Spy** меню.

 Доступны следующие параметры:

 **Инструмент поиска** перетаскивания средство, используемое для выбора окна.

 **Скрыть Spy ++** Удаляет главное окно Spy ++ и оставляет **найти окно** диалоговое окно поверх других приложений.

 **Обрабатывать** уникальный идентификатор окна для поиска.

 **Заголовок** текст заголовка окна.

 **Класс** класс окна.

 **Стиль** стиль окна.

 **Rect** ограничивающий прямоугольник для окна.

 **Показать свойства или сообщения** определяет, отображаются ли нажатие ОК [диалоговое окно "Свойства окна"](../debugger/window-properties-dialog-box.md) или откройте [представления сообщений](../debugger/messages-view.md).
