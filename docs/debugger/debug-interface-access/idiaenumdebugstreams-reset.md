# <a name="idiaenumdebugstreamsreset"></a>IDiaEnumDebugStreams::Reset
Сбрасывает последовательность перечисления до начала.

## <a name="syntax"></a>Синтаксис

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>Возвращаемое значение
 Возвращает `S_OK`.
