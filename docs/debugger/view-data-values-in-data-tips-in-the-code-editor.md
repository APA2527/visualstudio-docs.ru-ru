# <a name="view-data-values-in-datatips-in-the-code-editor"></a>Просмотр значений данных в подсказках по данным в редакторе кода

Подсказки данных предоставляют удобный способ просмотра сведений о переменных в программе во время отладки. Подсказки данных работают только в режиме прерывания и только с переменными, которые находятся в пределах текущей области выполнения. Если вы впервые пытались выполнить отладку кода, вам может потребоваться ознакомиться с [отладкой для начинающих](../debugger/debugging-absolute-beginners.md) , а также [методов и средств отладки](../debugger/write-better-code-with-visual-studio.md) , прежде чем переходить к этой статье.

## <a name="work-with-datatips"></a>Работа с подсказками по данным

Подсказки данных отображаются только в режиме приостановки выполнения и только для переменных, которые находятся в текущей области исполнения.

### <a name="display-a-datatip"></a>Отображение подсказки

1. Установите точку останова в коде и начните отладку, нажав клавишу **F5** или выбрав **Отладка** > **начать отладку**.

1. При приостановке в точке останова наведите указатель мыши на любую переменную в текущей области. Отобразится подсказка, в которой отображается имя и текущее значение переменной.

### <a name="make-a-datatip-transparent"></a>Сделать подсказку прозрачной

Чтобы сделать подсказку прозрачной для просмотра кода, находящегося под ней, в подсказке нажмите клавишу **CTRL**. Подсказка остается прозрачной, пока вы удерживаете нажатой клавишу **CTRL** . Это не работает для закрепленных или плавающих подсказок.
### <a name="pin-a-datatip"></a>Закрепление подсказки

Чтобы закрепить подсказку, чтобы она оставалась открытой, выберите значок канцелярская кнопка **в исходное** положение.

![Закрепление подсказки](../debugger/media/dbg-tips-data-tips-pinned.png "Закрепление подсказки")

Закрепленную подсказку можно переместить, перетащив ее вокруг окна кода. Значок канцелярской кнопки отображается во внутренней области рядом со строкой, на которую закреплена подсказка.

>[!NOTE]
>Подсказки данных всегда оцениваются в контексте, где выполнение приостановлено, а не текущего курсора или расположения подсказки. При наведении указателя мыши на переменную в другой функции, имя которой совпадает с именем переменной в текущем контексте, отображается значение переменной в текущем контексте.

### <a name="unpin-a-datatip-from-source"></a>Открепить подсказку из источника

Чтобы переместить закрепленную подсказку, наведите указатель мыши на подсказку и выберите значок канцелярской кнопки в контекстном меню.

Значок канцелярской кнопки изменится на незакрепленную, и теперь подсказка будет плавающей или может быть перенесена над всеми открытыми окнами. Плавающие подсказки закрываются при завершении сеанса отладки.

### <a name="repin-a-datatip"></a>Повторно закрепить подсказку

Чтобы повторно закрепить плавающую подсказку к источнику, наведите на нее указатель мыши в редакторе кода и щелкните значок канцелярской кнопки. Значок канцелярской кнопки изменится на закрепленную, а подсказка снова закрепляется только в окне кода.

Если всплывающая подсказка перемещается в неисходном окне кода, значок канцелярской кнопки недоступен, а подсказка не может быть перезакреплена. Чтобы получить доступ к значку канцелярской кнопки, верните подсказку в окно редактора кода, перетащив ее или передавая фокус окну кода.

### <a name="close-a-datatip"></a>Закрыть подсказку

Чтобы закрыть подсказку, наведите указатель мыши на подсказку и выберите значок Закрыть (**x**) в контекстном меню.

### <a name="close-all-datatips"></a>Закрыть все подсказки по данных

Чтобы закрыть все подсказки данных, в меню **Отладка** выберите команду **Очистить все подсказки**данных.

### <a name="close-all-datatips-for-a-specific-file"></a>Закрыть все подсказки по данным для определенного файла

Чтобы закрыть все подсказки по данным для определенного файла, в меню **Отладка** выберите команду **Очистить все подсказки данных, закрепленные в \<filename >** .

## <a name="expand-and-edit-information"></a>Развернуть и изменить сведения
Подсказки данных позволяют развернуть массив, структуру или объект для просмотра его элементов. Можно также изменить значение переменной из Подсказки Данных.

### <a name="expand-a-variable"></a>Развернуть переменную

Чтобы развернуть объект в подсказке для просмотра его элементов, наведите указатель мыши на стрелки развертывания перед именами элементов, чтобы отобразить элементы в виде дерева. Для закрепленной подсказки выберите **+** перед именем переменной, а затем разверните дерево.

![Развернуть подсказку](../debugger/media/dbg-tour-data-tips.png "Развернуть подсказку")

Можно использовать мышь или клавиши со стрелками на клавиатуре для перемещения вверх и вниз в развернутом представлении.

Можно также закрепить развернутые элементы в закрепленной подсказке, наведя на них указатель мыши и выбрав значки канцелярских кнопок. Элементы после сворачивания дерева отображаются в закрепленной подсказке.

### <a name="edit-the-value-of-a-variable"></a>Изменение значения переменной

Чтобы изменить значение переменной или элемента в подсказке, выберите значение, введите новое значение и нажмите клавишу **Ввод**. Выбор для значений, предназначенных только для чтения, отключен.

::: moniker range=">= vs-2019"

## <a name="pin-properties-in-datatips"></a>Закрепление свойств в подсказках данных

> [!NOTE]
> Эта функция поддерживается для .NET Core 3,0 или более поздней версии.

Вы можете быстро проверить объекты по их свойствам в подсказках по данным с помощью средства **также прикрепляемые Properties** Tool.  Чтобы использовать это средство, наведите указатель мыши на свойство и выберите значок закрепления или щелкните правой кнопкой мыши и в появившемся контекстном меню выберите пункт **закрепить элемент как избранный** .  Это помещает это свойство в верхнюю часть списка свойств объекта, а имя и значение свойства отображается в правом столбце подсказки.  Чтобы открепить свойство, щелкните значок закрепления еще раз или выберите пункт **закрепить член как избранный** в контекстном меню.

![Закрепление свойства в подсказке](../debugger/media/basic-pin-datatip.gif "Закрепление свойства в подсказке")

Можно также переключать имена свойств и отфильтровывать незакрепленные свойства при просмотре списка свойств объекта в подсказке.  Чтобы получить доступ к любому из этих вариантов, щелкните правой кнопкой мыши строку, содержащую свойство, и выберите пункт **Показывать только закрепленные элементы** или **Скрыть имена закрепленных членов в** параметрах значений в контекстном меню.

::: moniker-end

## <a name="visualize-complex-data-types"></a>Визуализация сложных типов данных

Значок лупы рядом с переменной или элементом в подсказке указывает, что для переменной доступны один или несколько [визуализаторов](../debugger/create-custom-visualizers-of-data.md), таких как [Визуализатор текста](../debugger/string-visualizer-dialog-box.md). Визуализаторы отображают сведения более информативным, иногда графическим способом.

Чтобы просмотреть элемент с помощью визуализатора по умолчанию для типа данных, выберите значок лупы значок ![визуализатора](../debugger/media/dbg-tips-visualizer-icon.png "Значок визуализатора"). Щелкните стрелку рядом со значком лупы, чтобы выбрать из списка визуализаторов для типа данных.

## <a name="add-a-variable-to-a-watch-window"></a>Добавление переменной в окно контрольных значений

Если вы хотите продолжить просмотр переменной, ее можно добавить в окно **контрольных значений** из подсказки. Щелкните правой кнопкой мыши переменную в подсказке и выберите **Добавить контрольное значение**.

Переменная появится в окне **Контрольные значения** . Если в выпуске Visual Studio поддерживается более одного окна **контрольных значений** , переменная появится в **контрольном списке 1**.

## <a name="import-and-export-datatips"></a>Советы по импорту и экспорту данных

Подсказки данных можно экспортировать в XML-файл, который можно совместно использовать или изменить в текстовом редакторе. Вы также можете импортировать XML-файл подсказки, который вы получили или редактировали.

**Экспорт подсказок данных:**

1. Выберите **отладка** > **Экспорт подсказок по**данных.

1. В диалоговом окне **Экспорт подсказок** перейдите в расположение, чтобы сохранить XML-файл, введите имя файла и нажмите кнопку **сохранить**.

**Импорт подсказок данных:**

1. Щелкните **отладка** > **Импорт подсказок**.

1. В диалоговом окне **Импорт подсказок** выберите XML-файл подсказок, который нужно открыть, и нажмите кнопку **Открыть**.
