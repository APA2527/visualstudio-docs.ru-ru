# <a name="window-menu-commands"></a>Команды меню "Окно"
**Окно** меню доступны команды для управления окнами Spy ++.

 **CASCADE** упорядочение окон каскадом.

 **Слева направо** упорядочение окон горизонтально.

 **Сверху вниз** упорядочение окон вертикально.

 **Упорядочить значки** упорядочение значков представлений, в том случае, если окна свернуты.

 **Закрыть / Закрыть все** закрывает один или все окна в Spy ++.

 **Обновить** обновление текущего выбранного представления.

 **Окно** возможность выбора отдельного окна.
