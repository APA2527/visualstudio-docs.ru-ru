# <a name="window-menu-commands"></a>Команды меню "Окно"
В меню **Окно** доступны команды для управления окнами Spy++.

 **Каскадом** позволяет упорядочить все окна каскадом.

 **Сверху вниз** позволяет переупорядочить все окна сверху вниз.

 **Слева направо** позволяет переупорядочить все окна слева направо.

 **Упорядочить значки** позволяет переупорядочить значки представлений, если окна свернуты.

 Параметры **Закрыть** или "Закрыть все" позволяют закрыть одно или все окна в Spy++.

 **Обновить** обновляет выбранное в этот момент представление.

 **Окно** позволяет выбрать конкретное окно.
