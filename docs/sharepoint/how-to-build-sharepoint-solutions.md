# <a name="how-to-build-sharepoint-solutions"></a>Руководство. построение решений SharePoint

Создание решения SharePoint не отличается от создания других приложений в Visual Studio. Дополнительные сведения о создании приложений см. [в разделе Компиляция и сборка в Visual Studio](../ide/compiling-and-building-in-visual-studio.md). Обсуждение некоторых аспектов, относящихся к решениям SharePoint, см. [в статье Развертывание и публикация решения SharePoint на локальном сайте SharePoint](../sharepoint/how-to-deploy-and-publish-a-sharepoint-solution-to-a-local-sharepoint-site.md).

## <a name="to-build-sharepoint-solutions"></a>Создание решений SharePoint

1. Решите, выполняется ли сборка для отладки или выпуска. Дополнительные сведения см. в разделе [Параметры проекта отладчика](../debugger/debugger-project-settings.md).

2. Выберите подходящий вариант для сборки или перестроения решения или проекта. Дополнительные сведения см. [в разделе Компиляция и сборка в Visual Studio](../ide/compiling-and-building-in-visual-studio.md).

3. В строке меню последовательно выберите **Сборка** > **Собрать решение**.
