# <a name="how-to-add-a-feature-to-sharepoint-solutions"></a>Как добавить компонент в решения SharePoint
  При создании решения SharePoint в решение добавляется одна функция и один пакет. Можно добавить дополнительные элементы компонентов для организации различных типов файлов SharePoint.

### <a name="to-add-a-feature-through-solution-explorer"></a>Добавление компонента с помощью обозреватель решений

1. В **Обозреватель решений** откройте контекстное меню **компонентов** и выберите пункт **Добавить компонент**.
