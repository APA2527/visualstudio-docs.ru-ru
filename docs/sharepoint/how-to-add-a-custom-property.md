# <a name="how-to-add-a-custom-property"></a>Как добавить пользовательское свойство
  Можно добавить пользовательские свойства в модель с помощью **редактора свойств**. Вы можете получить доступ к этим свойствам в коде, чтобы получить такие сведения, как строки подключения и другие данные во время выполнения.

### <a name="to-add-a-custom-property"></a>Добавление пользовательского свойства

1. В **обозревателе BDC**выберите узел, представляющий элемент модели, к которому необходимо применить пользовательское свойство.

2. В строке меню выберите **вид**  >  **окно свойств**.

3. В окне **Свойства** выберите свойство **пользовательские свойства** , а затем нажмите кнопку с многоточием (![ASP.net Mobile Designer Ellipse](../sharepoint/media/mwellipsis.gif "Эллипс конструктора ASP.NET для мобильных устройств")).

     Откроется диалоговое окно **Редактор свойств** .

4. В текстовом поле в столбце **имя** укажите имя свойства.

5. В поле **тип** пользовательского свойства выберите соответствующий тип данных.

6. В поле **значение** пользовательского свойства укажите значение, а затем нажмите кнопку **ОК** .
