>[!NOTE]
> Можно также выполнить поиск шаблонов, введя **SharePoint** в текстовом поле Поиск в верхней части диалогового окна **Создание нового проекта** . Можно также отфильтровать список шаблонов для отображения только шаблонов для Office и SharePoint, выбрав пункт "Office" в раскрывающемся списке Тип проекта. Дополнительные сведения см. [в разделе Создание нового проекта в Visual Studio](../../ide/create-new-project.md).
