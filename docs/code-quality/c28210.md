# <a name="c28210"></a>C28210
предупреждение 28210: Аннотации для контекста `_On_failure_` не должны быть в явном предварительном контексте

 Аннотации `_On_failure_` должны быть явно или неявно указаны в контексте `__post`, то есть применяться после возвращения функции.  Чтобы убедиться в этом, используйте `_drv_out`.
