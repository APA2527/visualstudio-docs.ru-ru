# <a name="c28222"></a>C28222
предупреждение 28222: \_да\_, \_нет\_или \_, возможно, ожидается\_ для аннотации

 Это предупреждение означает, что параметр в заметке должен быть одним из символов `_Yes_`, `_No_`или `_Maybe_`, а также обнаружен другой символ. Обычно это означает неправильное написание макроса аннотации.
