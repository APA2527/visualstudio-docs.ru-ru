# <a name="c6995"></a>C6995
C6995 предупреждения: Не удалось сохранить XML-файл журнала

 Это предупреждение означает, что средству анализа кода не удается создать журнал ошибок, который является выходом анализа кода.

 Эта ошибка может означать ошибку диска или указать, что у вас нет разрешения на создание файла в указанном каталоге.
