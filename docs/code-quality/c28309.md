# <a name="c28309"></a>C28309
предупреждение 28309: < parameter_name >: Операнды Аннотации должны быть типами Integer, Enum и Pointer. Операнды void и C++ перегруженные операторы не поддерживаются. Числа с плавающей запятой приблизительны на целые значения. Типы: \<typelist >.

 Вы попытались использовать функцию void или в выражении аннотации, и анализ кода не сможет его справиться.  Эта ошибка обычно возникает, когда используется `operator==`, реализованный как функция, но также могут возникать и другие случаи. Изучите типы в \<typelist >, чтобы объяснить, что не так.
