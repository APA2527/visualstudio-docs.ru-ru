# <a name="c28227"></a>C28227
C28227 предупреждения: Аннотация не может также быть PrimOp (см. предыдущее объявление)

 Это предупреждение означает, что именованная Аннотация объявляется как обычная Аннотация, а также ранее была объявлена как PrimOp. Это не должно быть возможным, если стандартные макросы используются для заметок; Это предупреждение не ожидается в типичном использовании.
