# <a name="c28138"></a>C28138
C28138 предупреждения: Константный аргумент должен быть переменным

 Это предупреждение выводится в вызове функции, ожидающем переменную или неконстантное выражение, но вызов включает константу. Сведения о функции и ее параметре см. в документации по функции WDK.

## <a name="example"></a>Пример
 Например, в следующем примере кода параметр макроса `READ_PORT_UCHAR` должен быть указателем на адрес порта, а не от адреса, предоставленного как константа.

 В следующем примере кода создается это предупреждающее сообщение:

```
READ_PORT_UCHAR(0x80001234);
```

 Чтобы исправить это предупреждение, используйте указатель на адрес порта.

```
READ_PORT_UCHAR(PortAddress);
```

 Существует несколько старых устройств, для которых параметр Constant допустим с семейством функций READ_PORT и WRITE_PORT. Когда эти устройства получают это предупреждение, предупреждение можно подавлять или игнорировать. Однако все новые устройства не должны считаться постоянным аппаратным адресом.
