# <a name="c28275"></a>C28275
Warning C28275: параметр для \_Macro \_value \_ имеет значение null

 Это предупреждение означает, что в файле модели имеется внутренняя ошибка, а не анализируемый код. Функция *макровалуе* была вызвана без параметра.
