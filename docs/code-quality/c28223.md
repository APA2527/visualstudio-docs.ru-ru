# <a name="c28223"></a>C28223
C28223 предупреждения: Не найден ожидаемый токен/идентификатор для аннотации, параметр

 Это предупреждение означает, что параметр в заметке должен быть идентификатором, а обнаружен другой символ. Обычно это означает неправильное написание макроса аннотации. Использование C или C++ ключевых слов в этой должности вызовет эту ошибку.
