# <a name="c28302"></a>C28302
C28302 предупреждения: Для C++ параметра reference-Parameter < parameter_name > обнаружен дополнительный оператор `_Deref_` в > \<annotation.

 Это предупреждение появляется, когда для параметра используется дополнительный уровень `_Deref_`.

 АННОТАЦИЕЙ SAL2 не требует использования дополнительного уровня `_Deref_` при работе со ссылочными параметрами. Эта заметка является неоднозначной и интерпретируется правильно, но ее следует исправить.

 Часто это можно исправить, просто удалив старую аннотацию `__deref` и используя синтаксис АННОТАЦИЕЙ SAL2. Иногда может потребоваться использовать `_At_` для ссылки на конкретный объект для аннотации.
