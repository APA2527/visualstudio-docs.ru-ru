# <a name="c28163"></a>C28163
C28163 предупреждения: Функция никогда не должна вызываться из блока try/except

 Это предупреждение выводится, если функция принадлежит к типу, который никогда не должен быть заключен в блок `try/except` в блоке `try/except`. Средство анализа кода обнаружило по крайней мере один путь, в котором функция была вызвана в блоке `try/except`.
