# <a name="c6510"></a>C6510
C6510 предупреждения: Недопустимая аннотация: Свойство "NullTerminated" может использоваться только для буферов, элементы которых имеют тип "Интеграл" или "pointer": Функция "\<function >" @no__t 1parameter >.

 Это предупреждение указывает на неправильное использование свойства **NullTerminated** (которое заканчивается на "`_z`"). Этот тип свойства можно использовать только в типах указателей или массивов.

## <a name="example"></a>Пример
 Это предупреждение создается в следующем коде:

```
#include <sal.h>

void f(_In_z_ char x)
{
    // code ...
}
```

 Чтобы исправить это предупреждение, используйте следующий код:

```
#include <sal.h>

void f(_In_z_ char * x)
{
    // code ...
}
```
