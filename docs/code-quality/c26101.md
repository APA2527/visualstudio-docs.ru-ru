# <a name="c26101"></a>C26101
C26101 предупреждения: Не удается правильно использовать блокируемую операцию для переменной \<var >.

 Интерфейсы API Windows предлагают разнообразные операции блокировки. Аннотация `_Interlocked_` указывает, что доступ к переменной должен осуществляться только через блокируемую операцию. Предупреждение C26101 выдается, когда доступ не согласуется с аннотацией `_Interlocked_`.

## <a name="example"></a>Пример
 В следующем примере создается предупреждение C26101, так как возникает нарушение контракта `_Interlocked_`.

```

CRITICAL_SECTION cs;
typedef struct _DATA
{
    _Interlocked_ LONG data;
} DATA;

void Safe(DATA* p)
{
    InterlockedIncrement(&p->data); // OK
}

void Unsafe(DATA* p)
{
    p->data += 1; // Warning C26101
    EnterCriticalSection(&cs);
    LeaveCriticalSection(&cs);
}
```
