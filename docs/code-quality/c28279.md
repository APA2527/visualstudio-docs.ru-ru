# <a name="c28279"></a>C28279
Warning C28279: для Symbol обнаружено "Begin без соответствующего" End "

 Язык аннотации поддерживает конструкцию Begin и End (`{` и `}` в C), а связывание стало несбалансированным. Эту ситуацию можно избежать, если используются макросы.
