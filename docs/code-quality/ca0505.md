# <a name="ca0505"></a>CA0505

Нерекомендуемое свойство игнорируется, так как оно определено более приоритетное свойство.

Предупреждение СА505 создается в том случае, если оба свойства анализа кода не рекомендуется и заменяющего значения которых лежат в проект или файл конфигурации сборки. Чтобы устранить эту проблему, удалите свойство не рекомендуется.
