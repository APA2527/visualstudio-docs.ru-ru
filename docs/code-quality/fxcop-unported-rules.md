# <a name="unported-rules"></a>Неперенесенные правила

## <a name="overview"></a>Обзор

Набор правил анализа прежних версий FxCop, которые не были реализованы как анализаторы Roslyn, состоит из следующих двух контейнеров:
- Правила, которые еще не были перенесены, но по-прежнему [могут быть перенесены](fxcop-unported-rules-may-get-ported.md).
- Устаревшие правила, которые [не будут перенесены](fxcop-unported-deprecated-rules.md).
