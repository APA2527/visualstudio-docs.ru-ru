# <a name="c28228"></a>C28228
C28228 предупреждения: Параметр аннотации: невозможно использовать тип в аннотации

 Это предупреждение означает, что параметр имеет тип, который не поддерживается. Заметки могут использовать только ограниченный набор типов в качестве параметров. Это не должно быть возможным, если стандартные макросы используются для заметок; Это предупреждение не ожидается в типичном использовании.
