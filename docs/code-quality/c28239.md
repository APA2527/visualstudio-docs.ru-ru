# <a name="c28239"></a>C28239
C28239 предупреждения: Аннотация в параметре одновременно содержит значение value, stringValue или longValue и выражение paramn=xxx. Используйте только paramn=xxx

 В файле быстрой модели произошла внутренняя ошибка. Это предупреждение не должно возникать при обычном использовании.
