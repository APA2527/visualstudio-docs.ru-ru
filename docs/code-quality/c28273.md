# <a name="c28273"></a>C28273
C28273 предупреждения: Функция: причины не согласуются с объявлением функции

 Это предупреждение указывает на ошибку в заметках, а не в анализируемом коде. Заметки, появляющиеся в определении функции, не согласуются с теми, которые отображаются в объявлении. Две заметки должны быть разрешены для сопоставления.
