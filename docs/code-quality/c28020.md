# <a name="c28020"></a>C28020
Warning C28020: выражение \<expr > не равно true в этом вызове

 Это предупреждение выводится в том случае, если выражение \_Satisfies \_ выражено неверно. Часто это указывает на неверный параметр.

 Если это происходит в объявлении функции, заметки указывают на невозможность условия.
