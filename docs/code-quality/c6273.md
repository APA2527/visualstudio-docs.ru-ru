# <a name="c6273"></a>C6273
Предупреждение 6273-нецелочисленное значение передано в качестве параметра \<number > Если требуется целое число при вызове \<function >.

 Это предупреждение означает, что в строке формата указано целое число, например `%d`, длина или спецификация приоритета для `printf`, но в качестве параметра передается нецелочисленное значение, такое как `float`, String или `struct`. Эта ошибка, скорее всего, приведет к неверным результатам.

## <a name="example"></a>Пример
 В следующем коде создается это предупреждение, поскольку вместо `float` в `sprintf`function требуется целое число:

```
#include <stdio.h>
#include <string.h>

void f_defective()
{
  char buff[50];
  float f=1.5;

  sprintf(buff, "%d",f);
}
```

 В следующем коде используется целочисленное приведение для исправления этого предупреждения:

```
#include <stdio.h>
#include <string.h>

void f_corrected()
{
  char buff[50];
  float f=1.5;

  sprintf(buff,"%d",(int)f);
}
```

 В следующем коде используется функция безопасной обработки строк, `sprintf_s`, чтобы исправить это предупреждение:

```
#include <stdio.h>
#include <string.h>

void f_safe()
{
  char buff[50];
  float f=1.5;

  sprintf_s(buff,50,"%d",(int)f);
}
```

 Это предупреждение неприменимо в Windows 9x и Windows NT версии 4, так как на этих платформах не поддерживается% p.
