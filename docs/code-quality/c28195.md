# <a name="c28195"></a>C28195
C28195 предупреждения: Функция объявлена как получающая память в переменной и завершила работу без этого

 Это предупреждение означает, что прототип функции для анализируемой функции имеет заметку `__drv_acquiresMemory`. Аннотация `__drv_acquiresMemory` указывает, что функция получает память в указанном расположении результатов, но по крайней мере в одном пути функция не получала память. Обратите внимание, что средство анализа кода не распознает фактическую реализацию распределителя памяти (включающее арифметические действия с адресом) и не распознает выделенную память (хотя многие оболочки будут распознаны). В этом случае средство анализа кода не распознает, что память была выделена, и выдает это предупреждение. Чтобы отключить ложное срабатывание, используйте предупреждение `#pragma` в строке, предшествующей открывающей фигурной скобке `{` тела функции.
