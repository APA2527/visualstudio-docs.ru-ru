# <a name="c28351"></a>C28351
C28351 предупреждения: Аннотация \<annotation > описывает, где динамическое значение (переменная) нельзя использовать в условии.

 Это предупреждение выводится при применении заметки при проверке типа C/C++ .

 Выражение в `_When_` должно оцениваться как константа. @No__t-0 игнорируется.
