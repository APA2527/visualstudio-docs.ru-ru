# <a name="c6289"></a>C6289
C6289 предупреждения: Неверный оператор: взаимное &#124; &#124; исключение не всегда равно ненулевой константе. Вы планируете использовать & &?

 Это предупреждение означает, что в тестовом выражении переменная проверяется на соответствие двум различным константам, а результат зависит от любого из условий, которые имеют значение true. Для этого всегда вычисляется значение true.

 Обычно эта проблема возникает при использовании `||` вместо `&&`, но также может быть вызвана использованием `!=`, где предполагалось `==`.

## <a name="example"></a>Пример
 Это предупреждение создается в следующем коде:

```
void f(int x)
{
  if ((x != 1) || (x != 3))
  {
    // code
  }
}
```

 Чтобы исправить это предупреждение, используйте следующий код:

```
void f(int x)
{
  if ((x != 1) && (x != 3))
  {
    // code
  }
}

/* or */
void f(int x)
{
  if ((x == 1) || (x == 3))
  {
    // code
  }
}
```
