# <a name="c28021"></a>C28021
C28021 предупреждения: Параметр \<param > закомментироваться с помощью \<anno > должен быть указателем

 Это предупреждение выводится в том случае, если объект с заметками не является типом указателя. Эту аннотацию нельзя использовать с `void` или целыми типами.
