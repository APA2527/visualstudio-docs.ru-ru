# <a name="c6305"></a>C6305
предупреждение C6305: возможное несоответствие количествам sizeof и каунтоф

 Это предупреждение означает, что переменная, содержащая результат `sizeof`, добавляется или вычитается из указателя или выражения `countof`. Это приведет к непредвиденному масштабированию в арифметике указателей.

## <a name="example"></a>Пример
 Это предупреждение создается в следующем коде:

```
void f(int *p)
{
  int cb=sizeof(int);
  //code...
  p +=cb; // warning 6305
}
```

 Чтобы исправить это предупреждение, используйте следующий код:

```
void f(int *p)
{
  // code...
  p += 1;
}
```
