# <a name="c28022"></a>C28022
C28022 предупреждения: Классы функций \<classlist1 > этой функции не совпадают с классами функций @no__t-> 1classlist2 в typedef, используемом для его определения.

 Это предупреждение появляется при возникновении ошибки в заметках. Как typedef, так и сама функция имеют `_Function_class_` аннотаций, но они не совпадают. Если используются оба значения, они должны совпадать.
