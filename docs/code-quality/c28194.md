# <a name="c28194"></a>C28194
C28194 предупреждения: Функция была объявлена как псевдоним значения в переменной и завершилась без этого

 Это предупреждение означает, что прототип функции для анализируемой функции имеет заметку `__drv_isAliased`, которая указывает, что он будет *псевдонимом* для указанного аргумента (т. е. присвоить значение таким образом, чтобы оно не сосохранило возврат из функции). Однако функция не применяет псевдоним аргумента вдоль пути, обозначенного заметкой. Большинство функций, которые имеют псевдоним для переменной, сохраняют свое значение в глобальной структуре данных.
