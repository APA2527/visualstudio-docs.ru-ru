# <a name="c28204"></a>C28204
Warning C28204: \<function >: только одна из этих перегрузок и \<filename > (\<line >) снабжаются заметками для \<paramname >: и то, и другое не должны быть снабжены заметками.

 Это предупреждение появляется при возникновении ошибки в заметках.
