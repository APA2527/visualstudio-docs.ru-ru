предупреждение C28204: \<function >: Только одна из этих перегрузок и одна из них в \<filename > (\<line >) снабжаются заметками для @no__t >-2paramname: и не должны закомментироваться.

 Это предупреждение появляется при возникновении ошибки в заметках.
