# <a name="c28204"></a>C28204
Предупреждение C28204: \<функция >: Только один из Эта перегрузка или перегрузка из \<filename > (\<строки >) аннотирована для \<paramname >: или не использующая оба должны быть аннотированы.

 Это предупреждение возникает, когда возникает ошибка в аннотации.
