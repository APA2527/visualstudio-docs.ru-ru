# <a name="c1252"></a>C1252
Предупреждение C1252: Циклическая или Отсутствующая зависимость между подключаемыми модулями: требуется GUID

 Средство анализа кода сообщает это предупреждение при возникновении внутренней ошибки с зависимостями подключаемого модуля, не в коде выполняется анализ.
