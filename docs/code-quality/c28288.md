# <a name="c28288"></a>C28288
предупреждение C28288: для функции, синтаксическая ошибка в \_At \_ () Аннотация (недопустимое имя параметра)

 Средство анализа кода сообщает об этом предупреждении, когда используется аннотация `SAL_at` (`__drv_at`) и выражение параметра не может быть интерпретировано в текущем контексте. Это может быть неправильное написание имени параметра или члена или неправильное использование ключевых слов "return" или "this".
