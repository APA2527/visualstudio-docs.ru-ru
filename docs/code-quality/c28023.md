# <a name="c28023"></a>C28023
C28023 предупреждения: Назначенная или передаваемая функция должна иметь заметку `_Function_class_` по крайней мере для одного из классов в: \<classlist >

 Это предупреждение обычно выводится, когда используется только один класс функции и обратный вызов соответствующего типа не объявлен.

 Это предупреждение выдается, когда функция в левой части назначения (или подразумеваемого присваивания, если это вызов функции) аннотирована, чтобы указать, что это тип функции, зависящей от драйвера, которая использует заметку `_Function_class_` или typedef, содержащий такой заметки. Функция в правой части назначения не имеет аннотации `_Function_class_`. Функция справа должна быть помечена тем же типом, что и функция слева. Обычно это рекомендуется сделать, добавив объявление \<class1 > @no__t-> 1funcname1 до текущего первого объявления \<funcname2 >.
