# <a name="c28215"></a>C28215
C28215 предупреждения: typefix нельзя применять к параметру, который уже имеет typefix

 Применение аннотации `__typefix` к параметру, который уже имеет эту аннотацию, является ошибкой. Заметки `__typefix` используются только в нескольких особых случаях, и это предупреждение не ожидается в обычном режиме.
