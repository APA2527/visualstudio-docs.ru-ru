# <a name="warning-c26453-arithmetic-overflow-left-shift-of-a-negative-signed-number-is-undefined-behavior"></a>C26453 предупреждения: Арифметическое переполнение: Сдвиг влево отрицательного числа со знаком является неопределенным поведением

Это предупреждение означает, что мы оставили сдвиг к отрицательному целочисленному значению со знаком, что является неправильной идеей и вызывает поведение, определенное реализацией.

## <a name="example"></a>Пример

```cpp
void leftshift(int shiftCount)
{
  const auto result = -1 << shiftCount;  // C26453 reported here

  // code
}
```

Чтобы исправить это предупреждение, используйте следующий код:

```cpp
void leftshift(int shiftCount)
{
  const auto result = ((unsigned)-1) << shiftCount; // OK

  // code
}
```
