# <a name="c28212"></a>C28212
C28212 предупреждения: Для аннотации ожидалось выражение указателя

 Это предупреждение означает, что нумерованный параметр в заметке (а не в аннотированной функции) должен быть указателем или типом массива, но обнаружен другой тип. Необходимо исправить заметку.
