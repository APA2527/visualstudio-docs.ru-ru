# <a name="c28211"></a>C28211
C28211 предупреждения: Для SAL_context ожидалось имя статического контекста

 Это предупреждение означает, что операндом для аннотации `_Static_context_` должно быть имя определенного в инструменте контекста. Это предупреждение никогда не должно возникать при обычном использовании.
