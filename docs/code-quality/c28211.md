# <a name="c28211"></a>C28211
предупреждение C28211: ожидалось имя статического контекста для SAL_context

 Это предупреждение означает, что операндом `_Static_context_` заметке должно быть имя определенного в инструменте контекста. Это предупреждение никогда не должно возникать при обычном использовании.
