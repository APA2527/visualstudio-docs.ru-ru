# <a name="ca0503"></a>CA0503
Свойство является устаревшим. Использование заменяющего свойства

Предупреждение CA0503 создается, когда нерекомендуемое свойство анализа кода содержит значение в сборке или файле конфигурации проекта. Чтобы устранить это предупреждение, используйте указанное свойство, которое заменяет его.
