# <a name="ca0503"></a>CA0503
Свойство является устаревшим. Заменяющее свойство

 CA0503 предупреждение создается в том случае, когда свойство анализа устаревший код содержит значение в файле конфигурации проекта или построения. Чтобы устранить это предупреждение, используйте указанное свойство, которое заменяет его.
