# <a name="c6272"></a>C6272
предупреждение C6272: параметр без типа float передается как аргумент \<number >, когда требуется float в вызове \<function >

 Это предупреждение означает, что строка формата указывает, что требуется float, например `%f` или `%g` для `printf,`, но передается не float, например, целое число или строка. Эта ошибка, скорее всего, приведет к неверным результатам. Однако в некоторых обстоятельствах это может привести к сбою.

## <a name="example"></a>Пример
 Это предупреждение создается в следующем коде:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %f","a",i);
}
```

 Чтобы исправить это предупреждение, используйте `%i` вместо спецификации `%f`, как показано в следующем коде:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf(buff,"%s %i","a",i);
}
```

 В следующем коде используется функция безопасной обработки строк, `sprintf_s`, чтобы исправить это предупреждение:

```
#include <stdio.h>
#include <string.h>

void f()
{
   char buff[5];
   int i=5;

   sprintf_s(buff,5,"%s %i","a",i); // safe version
}
```
