# <a name="c28024"></a>C28024
C28024 предупреждения: Указатель функции, которому присваивается значение, записывается в класс Function \<class >, который не содержится в классах функции \<classlist >.

 Это предупреждение выводится, когда обе функции были помечены классом функции, но классы не совпадают.

 Это предупреждение выдается, когда указатель функции имеет заметку `_Function_class_`, которая указывает, что ему должны быть назначены только функции определенного функционального класса. В присваивании или неявном назначении в вызове функции источник и цель должны быть одного класса функции, но классы функций не совпадают.
