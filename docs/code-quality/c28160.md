# <a name="c28160"></a>C28160
C28160 предупреждения: Ошибка аннотации

 Это предупреждение появляется при обнаружении аннотации `__drv_error`. Эта заметка используется для определения методов программирования, которые следует исправить, и может использоваться с аннотацией `__drv_when` для указания определенных сочетаний параметров.
