# <a name="c28160"></a>C28160
предупреждение C28160: Аннотация ошибки

 Это предупреждение появляется при обнаружении `__drv_error` заметки. Эта заметка используется для определения методов программирования, которые следует исправить, и может использоваться с заметкой `__drv_when` для указания определенных сочетаний параметров.
