# <a name="ca5361-do-not-disable-schannel-use-of-strong-crypto"></a>CA5361. Не отключайте стойкое шифрование в защищенном канале

|||
|-|-|
|TypeName|донотсетсвитч|
|CheckId|CA5361|
|Категория|Microsoft.Security|
|Критическое изменение|Не критическое|

## <a name="cause"></a>Причина:

При вызове `Switch.System.Net.DontEnableSchUseStrongCrypto`методаустанавливается значение `true`. <xref:System.AppContext.SetSwitch%2A?displayProperty=nameWithType>

## <a name="rule-description"></a>Описание правила

Параметр `Switch.System.Net.DontEnableSchUseStrongCrypto` , `true` чтобы ослабить шифрование, используемое в исходящих подключениях протокола TLS. Более слабая криптография может нарушить конфиденциальность взаимодействия между приложением и сервером, что облегчит злоумышленникам еавесдроп конфиденциальные данные. Дополнительные сведения см. в статье рекомендации по [обеспечению безопасности транспортного уровня (TLS) в .NET Framework](/dotnet/framework/network-programming/tls#switchsystemnetdontenableschusestrongcrypto).

## <a name="how-to-fix-violations"></a>Устранение нарушений

- Если приложение предназначено для .NET Framework v 4.6 или более поздней версии, можно <xref:System.AppContext.SetSwitch%2A?displayProperty=nameWithType> либо удалить вызов метода, либо задать для `false`параметра значение.
- Если приложение предназначено .NET Framework более ранней, чем версия 4.6 и выполняется на .NET Framework версии 4.6 или более поздней, задайте `false`для параметра значение.
- В противном случае ознакомьтесь с рекомендациями по [обеспечению безопасности транспортного уровня (TLS) с .NET Framework](/dotnet/framework/network-programming/tls) для устранения рисков.

## <a name="when-to-suppress-warnings"></a>Когда следует подавлять предупреждения

Это предупреждение можно отключить, если необходимо подключиться к устаревшей службе, которая не может быть обновлена для использования конфигураций Secure TLS.

```csharp

public class ExampleClass
{
    public void ExampleMethod()
    {

        AppContext.SetSwitch("Switch.System.Net.DontEnableSchUseStrongCrypto", true);

```

### <a name="solution"></a>Решение

```csharp
using System;

public class ExampleClass
{
    public void ExampleMethod()
    {
        AppContext.SetSwitch("Switch.System.Net.DontEnableSchUseStrongCrypto", false);
    }
}
```
