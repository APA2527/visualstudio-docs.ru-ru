# <a name="c28220"></a>C28220
Предупреждение C28220: Для параметра аннотации ожидалось целочисленное выражение

 Это предупреждение означает, что параметр заметки должен быть целочисленным выражением и обнаружен другого типа. Обычно это означает макроса неправильным кодом заметки.
