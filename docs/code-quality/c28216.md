# <a name="c28216"></a>C28216
C28216 предупреждения: Аннотация \_Check @ no__t-1return @ no__t-2 применяется только к последующим условиям для конкретного параметра функции.

 Аннотация `_Check_return_` применена в контексте, отличном от `__post`; может потребоваться модификатор `__post` (или `__drv_out`) или он может быть неправильно размещен.
