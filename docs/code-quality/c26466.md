# <a name="c26466-nostaticdowncastpolymorphic"></a>C26466 NO_STATIC_DOWNCAST_POLYMORPHIC
  Не используйте `static_cast` нисходящее приведение типа. Приведении полиморфного типа следует использовать dynamic_cast. См. в разделе [Type.2 правила C++ Core](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Pro-type-downcast).
