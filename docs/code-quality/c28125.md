# <a name="c28125"></a>C28125
C28125 предупреждения: Функция должна вызываться из блока try/except

 Драйвер вызывает функцию, которая должна вызываться из блока try/except, например [пробефорреад](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-probeforread), [пробефорврите](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-probeforwrite), [ммпробеандлоккпажес](https://docs.microsoft.com/windows-hardware/drivers/ddi/content/wdm/nf-wdm-mmprobeandlockpages).

## <a name="example"></a>Пример
 Следующий пример кода вызывает появление этого предупреждения.

```
ProbeForRead(addr, len, 4);
```

 Следующий пример кода позволяет избежать появления этого предупреждения.

```

__try
{
    ProbeForRead(addr, len, 4);
}
__except(EXCEPTION_EXECUTE_HANDLER)
{
    Status = GetExceptionCode();
    ... report error status
}
```
