# <a name="c6314"></a>C6314
C6314 предупреждения: Неверный порядок операций: побитовое или имеет более высокий приоритет, чем оператор условного выражения. Добавьте круглые скобки, чтобы уточнить смысл.

 Это сообщение означает, что выражение, содержащее оператор побитового или (`|`), обнаружено в проверяемом выражении условной операции (`?:`).

 Условный оператор имеет более низкий приоритет, чем побитовые операторы. Если проверяемое выражение должно содержать оператор побитового или, то вокруг условного выражения следует добавить круглые скобки.

## <a name="example"></a>Пример
 Это предупреждение создается в следующем коде:

```
int SystemState();

int f(int SignalValue)
{
  return SystemState() | (SignalValue != 0) ? 1 : 0;
}
```

 Чтобы исправить это предупреждение, используйте следующий код:

```
int SystemState();

int f(int SignalValue)
{
  return SystemState() | ((SignalValue != 0) ? 1 : 0);
}
```
