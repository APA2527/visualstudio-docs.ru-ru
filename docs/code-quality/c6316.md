# <a name="c6316"></a>C6316
C6316 предупреждения: Неверный оператор: проверяемое выражение является константой и не равно нулю. Используйте битовую операцию и, чтобы определить, установлены ли биты

 Это предупреждение указывает на использование битового или (`|`) при использовании побитового и (`&`). Побитовое или добавляет биты в результирующее выражение, в то время как побитовое и выбирает только те биты, которые являются общими между двумя операторами. Тесты для флагов должны выполняться с побитовой и проверкой на равенство.

## <a name="example"></a>Пример
 Это предупреждение создается в следующем коде:

```
#define INPUT_VALUE 2
void f( int Flags)
{
  if (Flags | INPUT_VALUE) // warning
  {
    // code
  }
}
```

 Чтобы исправить это предупреждение, используйте следующий код:

```
#define ALLOWED 1
#define INPUT_VALUE 2

void f( int Flags)
{
  if ((Flags & INPUT_VALUE) == ALLOWED)
  {
    // code
  }
}
```
