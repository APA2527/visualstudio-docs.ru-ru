# <a name="c28238"></a>C28238
Warning C28238: Аннотация для параметра имеет более одного значения, stringValue и longValue. Используйте paramn=xxx

 В файле быстрой модели произошла внутренняя ошибка. Это предупреждение не должно возникать при обычном использовании.
