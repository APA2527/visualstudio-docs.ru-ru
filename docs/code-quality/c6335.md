# <a name="c6335"></a>C6335
предупреждение C6335: дескриптор сведений о процессе утечки \<handlename >

 Это предупреждение означает, что дескрипторы сведений о процессе, возвращаемые семейством функций CreateProcess, необходимо закрыть с помощью CloseHandle. Несоблюдение этого действия приведет к утечке дескрипторов.

## <a name="example"></a>Пример
 Это предупреждение создается в следующем коде:

```
#include <windows.h>
#include <stdio.h>

void f( )
{
    STARTUPINFO si;
    PROCESS_INFORMATION pi;

    ZeroMemory( &si, sizeof(si) );
    si.cb = sizeof(si);
    ZeroMemory( &pi, sizeof(pi) );

    // Start the child process.
    if( !CreateProcess( "C:\\WINDOWS\\system32\\calc.exe",
                        NULL,
                        NULL,
                        NULL,
                        FALSE,
                        0,
                        NULL,
                        NULL,
                        &si,    // Pointer to STARTUPINFO structure.
                        &pi ) ) // Pointer to PROCESS_INFORMATION
  {
    puts("Error");
    return;
  }
    // Wait until child process exits.
    WaitForSingleObject( pi.hProcess, INFINITE );
    CloseHandle( pi.hProcess );
}
```

 Чтобы исправить это предупреждение, вызовите `CloseHandle (pi.``hThread)`, чтобы закрыть обработчик потока, как показано в следующем коде:

```
#include <windows.h>
#include <stdio.h>

void f( )
{
    STARTUPINFO si;
    PROCESS_INFORMATION pi;

    ZeroMemory( &si, sizeof(si) );
    si.cb = sizeof(si);
    ZeroMemory( &pi, sizeof(pi) );

    // Start the child process.
    if( !CreateProcess( "C:\\WINDOWS\\system32\\calc.exe",
                        NULL,
                        NULL,
                        NULL,
                        FALSE,
                        0,
                        NULL,
                        NULL,
                        &si,    // Pointer to STARTUPINFO structure.
                        &pi ) ) // Pointer to PROCESS_INFORMATION
    {
      puts("Error");
      return;
    }

    // Wait until child process exits.
    WaitForSingleObject( pi.hProcess, INFINITE );
    CloseHandle( pi.hProcess );
    CloseHandle( pi.hThread );
}
```

 Дополнительные сведения см. в разделе [функция CreateProcess](http://go.microsoft.com/fwlink/?LinkId=181206) и [функция CloseHandle](http://go.microsoft.com/fwlink/?LinkId=181207).
