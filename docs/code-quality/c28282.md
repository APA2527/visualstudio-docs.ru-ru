# <a name="c28282"></a>C28282
C28282 предупреждения: Строки формата должны находиться в предусловиях

 Это предупреждение означает, что обнаружена Аннотация `__drv_formatString`, которая не находится в аннотации `_Pre_` (`__drv_in`) (параметры функции по умолчанию имеют значение `_Pre_`). Проверьте, используется ли заметка в явном блоке с заметкой `_Post_` (`__drv_out`). Если это так, удалите заметку из любого вмещающего блока, который поместил его в контекст `_Post_`.
