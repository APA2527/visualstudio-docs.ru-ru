# <a name="ca0053"></a>CA0053
Не удалось загрузить сборку правил.

 Это означает, что не удалось загрузить сборку правил с помощью отражения. Это может произойти, если пользователь имеет настраиваемые правила, которые не были правильно созданы или средство анализа не удается загрузить сборку по другим причинам, таким образом, как был запрещен доступ к файловой системе.
