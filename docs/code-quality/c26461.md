# <a name="c26461-use_const_pointer_arguments"></a>C26461 USE_CONST_POINTER_ARGUMENTS:
  Аргумент указателя "% Argument%" для функции "% Function%" может быть помечен как указатель на `const`. См [ C++ . раздел Основные рекомендации Con. 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#Rconst-ref).
