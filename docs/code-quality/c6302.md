# <a name="c6302"></a>C6302
Предупреждение C6302: несоответствие строки формата: символьная строка передается как параметр \<number >, когда требуется строка расширенных символов в вызове \<function >

 Это предупреждение означает, что строка формата указывает, что строка расширенных символов является обязательной. Однако передается строка символов. Эта ошибка, скорее всего, приведет к сбою или повреждению некоторой формы.

## <a name="example"></a>Пример
 В следующем примере кода создается это предупреждение, так как в функцию `wprintf` передается символьная строка:

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%s", buff);
}
```

 В следующем примере кода используется `%hs` для указания однобайтовой строки символов с функцией `wprintf`:

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf(L"%hs", buff);
}
```

 В следующем примере кода используется функция безопасной обработки строк `wprintf_s` для устранения этого предупреждения:

```
#include<stdio.h>

void f()
{
  char buff[5] = "hi";

  wprintf_s(L"%hs", buff);
}
```
