# <a name="c28219"></a>C28219
C28219 предупреждения: Для параметра аннотации ожидался член перечисления

 Параметр в заметке должен быть членом именованного типа `enum`, а также обнаружен другой символ. Используйте член этого типа `enum`. Обычно это означает неправильное написание макроса аннотации.
