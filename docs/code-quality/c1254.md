# <a name="c1254"></a>C1254
предупреждение C1254: несоответствие версии подключаемого модуля: версия не соответствует версии предупреждающего драйвера

 Средство анализа кода сообщает об этом предупреждении при возникновении внутренней ошибки в версии подключаемого модуля, а не в анализируемом коде.
