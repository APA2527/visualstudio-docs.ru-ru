# <a name="c28301"></a>C28301
Warning C28301: нет аннотаций для первого объявления > \<function. \<note1 > см. \<filename > (\<line >). \<note2 >

 Это предупреждение выводится, когда заметки не были найдены при первом объявлении данной функции.
