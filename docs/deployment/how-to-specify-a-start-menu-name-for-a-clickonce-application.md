# <a name="how-to-specify-a-start-menu-name-for-a-clickonce-application"></a>Практическое руководство. Указание названия приложения ClickOnce в меню "Пуск"
При [!INCLUDE[ndptecclick](../deployment/includes/ndptecclick_md.md)] установке приложения как в интерактивном, так и в автономном режиме в меню " **Пуск** " и в списке " **Установка и удаление программ** " добавляется запись. По умолчанию отображаемое имя совпадает с именем сборки приложения, но вы можете изменить отображаемое имя, указав **имя продукта** в диалоговом окне **Параметры публикации** .

 **Название продукта** будет отображаться на *publish.htm* странице. для установленного автономного приложения это будет имя записи в меню " **Пуск** ", а также имя, которое отображается в окне " **Установка и удаление программ**".

 **Имя издателя** будет отображаться на *publish.htm* странице выше **названия продукта**, а для установленного автономного приложения это также будет имя папки, содержащей значок приложения в меню " **Пуск** ".

 Ярлык или ссылка на приложение в меню "Пуск" создается в *%Аппдата%\микрософт\виндовс\старт мену\програмс \\<\> имя издателя*. Имя ярлыка или ссылки на приложение совпадает с именем продукта.

 Свойства **имя продукта** и **имя издателя** можно задать в диалоговом окне **Параметры публикации** , которое доступно на странице **Публикация** в **конструкторе проектов**.

### <a name="to-specify-a-start-menu-name"></a>Указание имени меню "Пуск"

1. Выберите проект в **обозревателе решений**, а затем в меню **Проект** щелкните **Свойства**.

2. Перейдите на вкладку **Публикация**.

3. Нажмите кнопку " **Параметры** ", чтобы открыть диалоговое окно " **Параметры публикации** ".

4. Щелкните **Описание**.

5. В диалоговом окне **Параметры публикации** введите имя, которое будет отображаться в поле **Название продукта**.

6. При необходимости можно ввести имя издателя в качестве **имени издателя**.
