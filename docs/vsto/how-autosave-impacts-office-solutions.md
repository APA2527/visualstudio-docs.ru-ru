# <a name="how-autosave-impacts-office-solutions"></a>Как автосохранение влияет на решения Office

Автосохранение — это функция для Excel, PowerPoint и Word, которая автоматически и постоянно сохраняет изменения. Если эта функция отключена, необходимо сохранить файлы вручную, чтобы сохранить изменения. Может потребоваться настроить решение Office таким образом, чтобы оно было беспрепятственно работать даже при включенном автосохранении. Дополнительные сведения см. в разделе [как автосохранение влияет на надстройки и макросы](/office/vba/library-reference/concepts/how-autosave-impacts-addins-and-macros). Дополнительные сведения о сохранении в целом см. в разделе [что такое Автосохранение?](https://support.office.com/en-US/article/What-is-AutoSave-6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5).

> [!NOTE]
> Автосохранение для Windows Desktop Word, Excel и PowerPoint было введено в 2017 и в настоящее время доступно для подписчиков Office 365. Если вы приобрели бессрочную лицензию на Office 2016 или более раннюю версию, у вас нет доступа к функции совместной разработки. (Excel Online, Excel для Android, Excel для iOS и Excel Mobile в магазине Windows поддерживают эту функцию.)
