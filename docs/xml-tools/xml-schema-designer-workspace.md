# <a name="xml-schema-designer-workspace-views"></a>Представления рабочей области конструктора схем XML

Конструктор XML-схем (конструктор XSD) является графическим инструментом, позволяющим просматривать XML-схемы. В дополнение к [обозревателю XML-схем](../xml-tools/xml-schema-explorer.md), позволяющему просматривать дерево XML-схемы и выполнять поиск, конструктор XSD предоставляет три представления, в которых можно более подробно ИЗУЧИТЬ схему XSD.

- **Начальное представление** — это точка запуска для конструктора XSD; в начальном представлении можно перейти к другим представлениям конструктора XSD и просмотреть сведения о наборе схем.
- **Представление графика** позволяет просматривать общие сведения о наборе схем и связях между узлами схемы.
- **Представление модели содержимого** предоставляет графическое представление сведений о локальных и глобальных узлах схемы, включая простые и сложные типы, элементы, группы, атрибуты и группы атрибутов.

Чтобы начать изучение интересующих вас узлов, необходимо добавить их в рабочую область. Рабочая область совместно используется всеми представлениями.

## <a name="add-nodes-to-the-workspace"></a>Добавление узлов в рабочую область

Добавление узлов в рабочую область можно осуществлять следующими способами:

- В разделе "сведения о наборе схем" [представления "начало](../xml-tools/start-view.md)" щелкните ссылку **Добавить** рядом с типом глобального узла.

- Перетащите глобальные узлы, узлы файлов и узлы пространства имен из **обозревателя XML-схем** в любое из трех представлений. Дополнительные сведения см. в разделе "перетаскивание и удаление узлов" в [обозревателе XML-схем](../xml-tools/xml-schema-explorer.md).

- Используйте контекстное меню (контекст щелчка правой кнопкой мыши) в **обозревателе XML-схем**. Дополнительные сведения см. в разделе [Контекстные меню](../xml-tools/context-menus-xml-schema-explorer.md).

- Выполните поиск в обозревателе XSD и нажмите кнопку **Добавить выделенные узлы в рабочую область** на панели сводки результатов. Дополнительные сведения см. [в разделе Поиск по набору схем](../xml-tools/searching-the-schema-set.md).

## <a name="switch-views"></a>Переключение представлений

Для переключения между представлениями воспользуйтесь одним из приведенных ниже вариантов:.

- Панелью инструментов конструктора схемы XSD.

- Контекстные меню в представлении модели содержимого и представлении графа (щелчок правой кнопкой мыши).

- Водяными знаками на странице начального представления или водяным знаком в пустом представлении модели содержимого или представлении графика.

- Сочетания клавиш: **ctrl**+**1** для представления "начало", **CTRL**+**2** для представления графика и **CTRL**+**3** — для представления модели содержимого.
