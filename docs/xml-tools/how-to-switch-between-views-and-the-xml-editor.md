# <a name="how-to-switch-between-views-and-the-xml-editor"></a>Руководство. Переключение между представлениями и редактором XML

В этом разделе показано, как переключаться между представлениями конструктора схем XML (XSD Designer) и редактором XML. В этом примере используется [Схема заказа на покупку](../xml-tools/sample-xsd-file-simple-schema.md).

## <a name="to-switch-between-the-views-and-the-xml-editor"></a>Переключение между представлениями и редактором XML

1. Чтобы создать и изменить новый файл схемы XML, выполните действия, описанные в разделе [Создание и изменение файла XSD](../xml-tools/how-to-create-and-edit-an-xsd-schema-file.md)-схемы.

2. Чтобы переключиться на конструктор XML-схем из редактора XML, щелкните правой кнопкой мыши в любом месте редактора XML и выберите пункт **Конструктор представлений**.

3. Чтобы переключиться на представление графа с помощью водяного знака, щелкните элемент **использовать представление графа, чтобы просмотреть связь между узлами** в начальном представлении.

4. Перетащите `USAddress` узел из **обозревателя XML-схем** в представление графа. Щелкните правой кнопкой мыши узел `USAddress` в представлении графика и выберите пункт **Показать в представлении модели содержимого** в контекстном меню.

     Откроется представление модели содержимого со сведениями об узле `USAddress`.

5. Чтобы переключиться в представление "начало" из представления модели содержимого с помощью панели инструментов, нажмите кнопку **начать просмотр** на панели инструментов XSD.

6. Чтобы переключиться между представлениями с помощью сочетаний клавиш, нажмите сочетание клавиш **ctrl**+**1** для представления "начало", **CTRL**+**2** в представлении графика и **CTRL**+**3** — для представления модели содержимого.

7. Чтобы перейти к редактору XML из представления модели содержимого, щелкните узел правой кнопкой мыши и выберите в контекстном меню пункт **Просмотреть код** .
