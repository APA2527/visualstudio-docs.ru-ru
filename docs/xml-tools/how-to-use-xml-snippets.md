# <a name="how-to-use-xml-snippets"></a>Как использовать XML-фрагменты

Фрагменты XML можно вызывать с помощью следующих двух команд в контекстном меню редактора XML. Команда **Вставить фрагмент** вставляет XML-фрагмент в позицию курсора. Команда **окружить с помощью** оболочки фрагмента кода XML вокруг выделенного текста. Для каждого XML-фрагмента имеются назначенные типы фрагментов. Типы фрагментов определяют, доступен ли фрагмент кода с помощью команды « **Вставить фрагмент** », команды « **окружить** » или и того, и другого.

После добавления XML-фрагмента в редактор все редактируемые поля во фрагменте выделяются желтым цветом, а курсор помещается в первое редактируемое поле.

## <a name="insert-snippet"></a>Вставить фрагмент

В следующих процедурах описывается, как получить доступ к команде **Вставить фрагмент** .

> [!NOTE]
> Команда **Вставить фрагмент** также доступна через сочетание клавиш (**CTRL** +**K**, а затем **CTRL** +**X**).

### <a name="to-insert-snippets-from-the-shortcut-menu"></a>Вставка фрагментов из контекстного меню

1. Поместите курсор в то место, куда нужно добавить XML-фрагмент.

2. Щелкните правой кнопкой мыши и выберите **Вставить фрагмент**.

   Отображается список доступных XML-фрагментов.

3. Выберите фрагмент из списка с помощью мыши или введите имя фрагмента и нажмите клавишу **Tab** или **Ввод**.

### <a name="to-insert-snippets-using-the-intellisense-menu"></a>Вставка фрагментов с помощью меню IntelliSense

1. Поместите курсор в то место, куда нужно добавить XML-фрагмент.

2. В меню **Правка** наведите указатель мыши на пункт **IntelliSense**и выберите пункт **Вставить фрагмент**.

   Отображается список доступных XML-фрагментов.

3. Выберите фрагмент из списка с помощью мыши или введите имя фрагмента и нажмите клавишу **Tab** или **Ввод**.

### <a name="to-insert-snippets-through-the-intellisense-complete-word-list"></a>Вставка фрагментов кода с помощью списка слов завершения IntelliSense

1. Поместите курсор в то место, куда нужно добавить XML-фрагмент.

2. Начните вводить XML-фрагмент, который нужно добавить в файл. Если включено автоматическое завершение, отобразится список слов IntelliSense для автоматического завершения. Если он не отображается, нажмите клавишу **Ctrl** +**пробел** , чтобы активировать его.

3. Выберите XML-фрагмент из списка автоматического завершения слов.

4. Нажмите клавишу **Tab**, **Tab** , чтобы вызвать XML-фрагмент.

> [!NOTE]
> В некоторых случаях XML-фрагмент не вызывается. К примеру, если попытаться вставить элемент `xs:complexType` внутри узла `xs:element`, редактор не создает XML-фрагмент. Когда элемент `xs:complexType` используется внутри узла `xs:element`, отсутствуют необходимые атрибуты или вложенные элементы, поэтому редактор не имеет данных для вставки.

### <a name="to-insert-snippets-using-the-shortcut-name"></a>Вставка фрагментов с помощью сокращенных имен

1. Поместите курсор в то место, куда нужно добавить XML-фрагмент.

2. Наберите `<` на панели редактора.

3. Нажмите клавишу **ESC** , чтобы закрыть список слов завершение работы IntelliSense.

4. Введите имя ярлыка фрагмента и нажмите клавишу **Tab** , чтобы вызвать XML-фрагмент.

## <a name="surround-with"></a>Окружить

В следующих процедурах описывается, как получить доступ к команде **окружить** .

> [!NOTE]
> Команда **окружить** также доступна через сочетание клавиш (**CTRL** +**K**, а затем **CTRL** +**S**).

### <a name="to-use-surround-with-from-the-context-menu"></a>Использование окружения с помощью контекстного меню

1. Выберите текст, который будет заключаться в редакторе XML.

2. Щелкните правой кнопкой мыши и выберите пункт **заключить в**.

   Отображается список XML-фрагментов, доступных для окружения.

3. Выберите фрагмент из списка с помощью мыши или введите имя фрагмента и нажмите клавишу **Tab** или **Ввод**.

### <a name="to-use-surround-with-from-the-intellisense-menu"></a>Использование функции окружить в меню IntelliSense

1. Выберите текст, который будет заключаться в редакторе XML.

2. В меню **Правка** наведите указатель мыши на пункт **IntelliSense**и выберите пункт **заключить в**.

   Отображается список XML-фрагментов, доступных для окружения.

3. Выберите фрагмент из списка с помощью мыши или введите имя фрагмента и нажмите клавишу **Tab** или **Ввод**.

## <a name="use-xml-snippets"></a>Использование фрагментов XML

Если выбран XML-фрагмент, текст фрагмента кода автоматически вставляется в текущей позиции курсора. Все редактируемые поля фрагмента выделяются цветом, а первое редактируемое поле выбирается автоматически. Выбранное в данный момент поле окружено рамкой.

Выделив поле, можно ввести для него новое значение. При нажатии клавиши **Tab** осуществляется перебор изменяемых полей фрагмента. Нажатие клавиш **Shift** +**Tab** циклически перебирает их в обратный порядок. Если щелкнуть поле один раз, курсор помещается в поле, а если щелкнуть поле дважды, оно будет выделено. Если поле выделено подсветкой, может отображаться подсказка с описанием этого поля.

Редактируемым является только первый экземпляр данного поля. После выделения такого поля прочие экземпляры поля обозначаются контуром. Если изменить значение в редактируемом поле, это поле изменится везде, где оно встречается во фрагменте.

Нажатие клавиши **Ввод** или **ESC** отменяет редактирование поля и возвращает редактор в нормальный режим.

Цвета по умолчанию для изменяемых полей фрагментов кода можно изменить, изменив параметр **поля фрагмент кода** в области **шрифты и цвета** диалогового окна **Параметры** . Дополнительные сведения см. [в разделе руководство. изменение шрифтов и цветов в редакторе](../ide/reference/how-to-change-fonts-and-colors-in-the-editor.md).
