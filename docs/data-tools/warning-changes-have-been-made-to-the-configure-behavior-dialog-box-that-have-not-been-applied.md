# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>Предупреждение. Имеются не примененные изменения диалогового окна "Настройка поведения"

Предупреждение. В диалоговом окне «Настройка режима» внесены изменения, которые еще не применены. Применить изменения?

Диалоговое окно **Настройка поведения** позволяет настроить поведение `Insert`, `Update` и `Delete` для всех доступных классов. Это сообщение появляется при выборе новой комбинации **Класса** и **Поведения**, когда предыдущее изменение еще не было применено.

## <a name="change-options"></a>Изменение параметров

- Чтобы применить изменение и продолжить, нажмите кнопку **Да**. Изменение применяется к выбранному **классу** и **поведению**.

- Чтобы отменить предыдущее изменение и продолжить, нажмите кнопку **нет**.
