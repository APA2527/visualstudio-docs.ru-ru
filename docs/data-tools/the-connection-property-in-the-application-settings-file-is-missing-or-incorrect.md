# <a name="the-connection-property-in-the-application-settings-file-is-missing-or-incorrect"></a>Свойство подключения в файле параметров приложения отсутствует или неверно

Свойства соединения в файле настройки приложения пропущены или неверны. В его месте используется строка подключения из *DBML* .

*DBML* -файл содержит ссылку на строку подключения в файле параметров приложения, которую не удается найти. Это сообщение является информационным; установка строки подключения будет создана при нажатии кнопки **OK**.

Чтобы ответить на это сообщение, нажмите кнопку **ОК**. Сведения о соединении, содержащиеся в файле *. dbml* , добавляются в параметры приложения.
