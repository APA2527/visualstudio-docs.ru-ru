> [!NOTE]
> Закодированный тест пользовательского интерфейса для автоматизированного функционального тестирования на основе интерфейса пользователя признан нерекомендуемым. Visual Studio 2019 является последней версией, в которой будет доступен закодированный тест пользовательского интерфейса. Мы рекомендуем использовать [Selenium](https://docs.seleniumhq.org/) для тестирования веб-приложений и [Appium с WinAppDriver](https://github.com/Microsoft/WinAppDriver) для тестирования классических приложений и приложений универсальной платформы Windows. Попробуйте воспользоваться [Xamarin.UITest](/appcenter/test-cloud/uitest/) для тестирования приложений iOS и Android с помощью платформы тестирования NUnit.
