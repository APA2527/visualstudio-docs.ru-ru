# <a name="control-flow-activity-designers"></a>Конструкторы действия управления потоком

Конструктор рабочих процессов включает в себя ряд предоставляемых системой действий, которые можно использовать для построения рабочих процессов. В этом разделе содержатся действия, предусмотренные системой, и служащие для управления потоком в пределах рабочего процесса. В следующих подразделах приводится описание этих действий, а также описание их использования.
