# <a name="messaging-activity-designers"></a>Конструкторы действий обмена сообщениями

Конструкторы операций обмена сообщениями используются для создания и настройки действий обмена сообщениями, отправки и получения сообщений Windows Communication Foundation (WCF) из приложения Windows Workflow Foundation (WF). Пять действий обмена сообщениями были введены в .NET Framework 4. Конструктор рабочих процессов предоставляет два шаблона конструктора, которые позволяют управлять обменом сообщениями в рабочем процессе.

Содержится в этом разделе и перечисленных в следующей таблице представлены рекомендации по использованию действий конструктора рабочих процессов и конструкторов шаблонов.

- <xref:System.Activities.Activity>

- <xref:System.ServiceModel.Activities.CorrelationScope>

- <xref:System.ServiceModel.Activities.Receive>

- <xref:System.ServiceModel.Activities.Send>

- <xref:System.ServiceModel.Activities.ReceiveReply>

- <xref:System.ServiceModel.Activities.SendReply>

- <xref:System.ServiceModel.Activities.TransactedReceiveScope>
