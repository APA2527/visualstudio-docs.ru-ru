# <a name="transaction-activity-designers"></a>Конструкторы действий транзакций

Конструкторы операций транзакций используются для создания и настройки действий транзакций, например, компенсации и подтверждения. Разделы, содержащиеся в этом разделе и перечисленные в следующей таблице, содержат рекомендации по использованию конструктор рабочих процессовных конструкторов действий и шаблонов.
