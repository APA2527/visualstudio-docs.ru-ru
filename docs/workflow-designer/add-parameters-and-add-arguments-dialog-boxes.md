# <a name="add-parameters-and-add-arguments-dialog-boxes"></a>Диалоговые окна «Добавление параметров» и «Добавление аргументов»

В следующей таблице описаны элементы пользовательского интерфейса (UI) **добавить параметры** диалоговое окно, используемое с <xref:System.Activities.Statements.InvokeMethod> конструктор:

|||
|-|-|
|**Направление**|Определяет, представляет параметр поток данных в метод, из метода или и тот и другой.|
|**Type**|Имя типа нового параметра.|
|**Значение**|Выражение Visual Basic, который используется для присвоения значения по умолчанию новому параметру|

В следующей таблице описаны элементы пользовательского интерфейса (UI) **Добавление аргументов** диалоговое окно, используемое с <xref:System.Activities.Statements.InvokeDelegate> конструктор:

|||
|-|-|
|**Name**|Имя аргумента.|
|**Направление**|Определяет, представляет аргумент поток данных в делегат, из делегата или и тот и другой.|
|**Type**|Имя типа нового аргумента.|
|**Значение**|Значение, используемое для этого экземпляра аргумента делегата.|
