# <a name="develop-apps-with-the-workflow-designer"></a>Разработка приложений с конструктором рабочих процессов

Конструктор рабочего процесса является визуальным дизайнером и отладчиком для графического строительства и отладки приложений [Windows Workflow Foundation](/dotnet/framework/windows-workflow-foundation/index) (WF) в Visual Studio. Это позволяет составить композитное приложение рабочего процесса, библиотеку активности или службу Windows Communication Foundation (WCF) с помощью шаблонов и дизайнеров деятельности.

## <a name="install-windows-workflow-foundation"></a>Установка Фонда рабочего процесса Windows

Чтобы использовать шаблоны проектов Workflow в Visual Studio, сначала установите компонент **Фонда рабочего процесса Windows.**

1. Открыть Visual Studio Установщик. Быстрый способ открыть его, выбрав **инструменты** > **Получить инструменты и особенности** в visual Studio.

1. В Visual Studio Installer выберите вкладку **Отдельные компоненты.**

1. Прокрутите вниз в категорию **действий разработки,** а затем выберите компонент **Фонда рабочего процесса Windows.**

   ![Компонент Фонда рабочего процесса Windows для визуальной студии](media/windows-workflow-foundation-component.png)

1. Выберите **Изменить**.

   Visual Studio устанавливает компонент **Фонда рабочего процесса Windows.**
