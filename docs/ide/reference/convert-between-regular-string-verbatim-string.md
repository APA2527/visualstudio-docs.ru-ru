# <a name="convert-between-regular-string-and-verbatim-string-literals-refactoring"></a>Преобразование между обычным и буквальным строковыми литералами (рефакторинг)

Область применения этого рефакторинга:

- C#

**Что?** Преобразование между обычным и буквальным строковыми литералами.

**Когда?** Вам нужно сэкономить место или сделать код более понятным.

**Зачем?** Преобразование между обычным и буквальным строковыми литералами помогает сэкономить место. Преобразование обычного строкового литерала в буквальный строковый литерал помогает сделать код более понятным.

## <a name="how-to"></a>Практические советы

1. Поместите курсор на обычный или буквальный строковый литерал.

2. Нажмите клавиши **CTRL**+ **.** чтобы открыть меню **Быстрые действия и рефакторинг**.

3. Выберите один из следующих вариантов:

    Выберите **Преобразовать в обычную строку**.

    ![Преобразовать в обычную строку](media/convert-to-regular-string.png)

    Выберите **Преобразовать в буквальную строку**.

    ![Преобразовать в буквальную строку](media/convert-to-verbatim-string.png)
