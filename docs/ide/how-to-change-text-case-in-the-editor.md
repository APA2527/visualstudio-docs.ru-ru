# <a name="how-to-change-text-case-in-the-editor"></a>Практическое руководство. изменение регистра текста в редакторе

Для переключения регистра выбранного текста можно использовать команды меню или сочетания клавиш.

> [!NOTE]
> Отображаемые диалоговые окна и команды меню могут отличаться от описанных в этой статье. Здесь приводятся **общие** параметры разработки. Чтобы изменить параметры разработки, **Сервис** > **Импорт и экспорт параметров**, а затем щелкните **Сбросить все параметры**.

## <a name="to-change-text-case"></a>Изменение регистра текста

1. Выделите текст, который требуется преобразовать.

2. Чтобы преобразовать все буквы в прописные, выберите **Изменить** > **Расширенные** > **Все прописные** или нажмите **CTRL**+**SHIFT**+**U**.

   Чтобы преобразовать все буквы в нижний регистр, выберите **Изменить** > **Расширенные** > **Нижний регистр** или нажмите **CTRL**+**U**.

> [!TIP]
> Чтобы вернуться к форматированию, предшествовавшему изменению, выберите **Отменить** в меню **Правка**.
