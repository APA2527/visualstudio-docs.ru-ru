# <a name="idebugengine2enumprograms"></a>IDebugEngine2::EnumPrograms
Получает список всех программ, отлаженных отладкой с помощью отладки двигателя (DE).

## <a name="syntax"></a>Синтаксис

```cpp
HRESULT EnumPrograms( 
   IEnumDebugPrograms2** ppEnum
);
```

```csharp
int EnumPrograms( 
   out IEnumDebugPrograms2 ppEnum
);
```

## <a name="parameters"></a>Параметры
`ppEnum`\
(ваут) Возвращает объект [IEnumDebugPrograms2,](../../../extensibility/debugger/reference/ienumdebugprograms2.md) содержащий список всех программ, отслусткоторых DE.

## <a name="return-value"></a>Возвращаемое значение
 Возвращает значение `S_OK`, если выполнение прошло успешно; в противном случае возвращает код ошибки.
