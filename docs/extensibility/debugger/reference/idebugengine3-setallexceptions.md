# <a name="idebugengine3setallexceptions"></a>IDebugEngine3::SetAllExceptions
Этот метод устанавливает состояние всех невыполненных исключений.

## <a name="syntax"></a>Синтаксис

```cpp
HRESULT SetAllExceptions(
   EXCEPTION_STATE dwState
);
```

```csharp
int SetAllExceptions(
   enum_EXCEPTION_STATE dwState
);
```

## <a name="parameters"></a>Параметры
`dwState`\
(в) Одним из [EXCEPTION_STATE](../../../extensibility/debugger/reference/exception-state.md) значений.

## <a name="return-value"></a>Возвращаемое значение
 В случае `S_OK`успеха, возвращается ; в противном случае возвращает код ошибки.
