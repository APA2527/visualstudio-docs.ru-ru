# <a name="idebugcoreserver3getconnectionprotocol"></a>IDebugCoreServer3::GetConnectionProtocol
Возвращает значение с указанием протокола, используемого для связи между сервером и пакетом отладки.

## <a name="syntax"></a>Синтаксис

```cpp
HRESULT GetConnectionProtocol(
   CONNECTION_PROTOCOL* pProtocol
);
```

```csharp
int GetConnectionProtocol(
   CONNECTION_PROTOCOL[] pProtocol
);
```

## <a name="parameters"></a>Параметры
`pProtocol`\
(ваут) Возвращает одно из значений из [CONNECTION_PROTOCOL](../../../extensibility/debugger/reference/connection-protocol.md) перечисления.

## <a name="return-value"></a>Возвращаемое значение
 В случае `S_OK`успеха, возвращается ; в противном случае возвращает код ошибки.
