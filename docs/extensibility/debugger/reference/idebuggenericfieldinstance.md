# <a name="idebuggenericfieldinstance"></a>IDebugGenericFieldInstance
Представляет экземпляр поля для общего типа управляемого кода.

## <a name="syntax"></a>Синтаксис

```
IDebugGenericFieldInstance : IUnknown
```

## <a name="methods"></a>Методы
 Этот интерфейс реализует следующие методы:

|Метод|Описание|
|------------|-----------------|
|[GetTypeArguments](../../../extensibility/debugger/reference/idebuggenericfieldinstance-gettypearguments.md)|Извлекает аргументы параметра типа для данного экземпляра.|
|[TypeArgumentCount](../../../extensibility/debugger/reference/idebuggenericfieldinstance-typeargumentcount.md)|Возвращает число аргументов параметра типа для данного экземпляра.|

 Заголовок: Sh.h

 Название: Microsoft.VisualStudio.Debugger.Interop

 Сборка: Microsoft.VisualStudio.Debugger.Interop.dll
