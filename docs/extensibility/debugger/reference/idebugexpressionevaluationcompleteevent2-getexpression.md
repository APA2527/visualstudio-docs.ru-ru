# <a name="idebugexpressionevaluationcompleteevent2getexpression"></a>IDebugExpressionEvaluationCompleteEvent2::GetExpression
Получает исходное выражение.

## <a name="syntax"></a>Синтаксис

```cpp
HRESULT GetExpression( 
   IDebugExpression2** ppExpr
);
```

```csharp
int GetExpression( 
   out IDebugExpression2 ppExpr
);
```

## <a name="parameters"></a>Параметры
`ppExpr`\
(ваут) Возвращает объект [IDebugExpression2,](../../../extensibility/debugger/reference/idebugexpression2.md) представляющий разогнанный выражение.

## <a name="return-value"></a>Возвращаемое значение
 Возвращает значение `S_OK`, если выполнение прошло успешно; в противном случае возвращает код ошибки.

## <a name="remarks"></a>Примечания
 Этот метод возвращает объект, созданный в вызове, к методу [ParseText.](../../../extensibility/debugger/reference/idebugexpressioncontext2-parsetext.md)
