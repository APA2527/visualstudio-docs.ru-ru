# <a name="idebugproperty2getmemorycontext"></a>IDebugProperty2::GetMemoryContext
Получает контекст памяти значения свойства.

## <a name="syntax"></a>Синтаксис

```cpp
HRESULT GetMemoryContext ( 
   IDebugMemoryContext2** ppMemory
);
```

```csharp
int GetMemoryContext(
   out IDebugMemoryContext2 ppMemory
);
```

## <a name="parameters"></a>Параметры
`ppMemory`\
(ваут) Возвращает объект [IDebugMemoryContext2,](../../../extensibility/debugger/reference/idebugmemorycontext2.md) представляющий память, связанную с этим свойством.

## <a name="return-value"></a>Возвращаемое значение
 В случае `S_OK`успеха, возвращается ; в противном случае возвращает код ошибки. Возвращается, `S_GETMEMORYCONTEXT_NO_MEMORY_CONTEXT` если нет контекста памяти для извлечения.
