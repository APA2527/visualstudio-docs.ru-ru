# <a name="machine_info_flags"></a>MACHINE_INFO_FLAGS
Используется для описания машины.

## <a name="syntax"></a>Синтаксис

```cpp
enum enum_MACHINE_INFO_FLAGS { 
   MCIFLAG_TERMINAL_SERVICES_AVAILABLE = 0x00000001
};
typedef DWORD MACHINE_INFO_FLAGS;
```

```csharp
public enum enum_MACHINE_INFO_FLAGS { 
   MCIFLAG_TERMINAL_SERVICES_AVAILABLE = 0x00000001
};
```

## <a name="fields"></a>Поля
 `MCIFLAG_TERMINAL_SERVICES_AVAILABLE`\
 Указывается, что терминалуслуги доступны.

## <a name="remarks"></a>Примечания
 Используется в `Flags` качестве члена [структуры MACHINE_INFO.](../../../extensibility/debugger/reference/machine-info.md)

 Заголовок: msdbg.h

 Название: Microsoft.VisualStudio.Debugger.Interop

 Сборка: Microsoft.VisualStudio.Debugger.Interop.dll
