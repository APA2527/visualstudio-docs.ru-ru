# <a name="idebugbindergetfunctionobject"></a>IDebugBinder::GetFunctionObject
Этот метод получает объект [IDebugFunctionObject,](../../../extensibility/debugger/reference/idebugfunctionobject.md) используемый для создания параметров функции.

## <a name="syntax"></a>Синтаксис

```cpp
HRESULT GetFunctionObject( 
   IDebugFunctionObject **ppFunction
);
```

```csharp
int GetFunctionObject(
   out IDebugFunctionObject ppFunction
);
```

## <a name="parameters"></a>Параметры
`ppFunction`\
(ваут) Возвращает интерфейс [IDebugFunction,](../../../extensibility/debugger/reference/idebugfunctionobject.md) который используется для создания параметров функции.

## <a name="return-value"></a>Возвращаемое значение
 В случае успеха, возвращается S_OK; в противном случае возвращает код ошибки.
