# <a name="list-of-available-services"></a>Список доступных служб

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]и Visual Studio SDK поддерживают следующие услуги. Некоторые пакеты предлагают свои собственные услуги, которые не перечислены здесь, например, языковые услуги не имеют единой службы GUID. Вы должны использовать имя языка, чтобы найти GUID языковой службы в реестре.

Используйте услуги GUID, перечисленные здесь или полученные из другого источника (например, языковые службы), чтобы получить основной интерфейс или интерфейсы, отображаемые с каждой службой.
