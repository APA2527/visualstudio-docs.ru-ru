# <a name="manifest-to-code"></a>Manifest to Code
Инструмент Manifest to Code — это консольное приложение, которое принимает файл .imagemanifest для Службы изображений Visual Studio и генерирует файл обертки или файлы для ссылки на значения манифеста изображения в C, C, VB или .vsct файлы для расширений Visual Studio. Этот инструмент генерирует файлы обертки, которые могут быть использованы для запроса изображений из Службы изображений Visual Studio напрямую или для передачи значений манифеста через AI, если код не обрабатывает ни один из своих собственных uI и рендеринга.

## <a name="how-to-use-the-tool"></a>Как использовать инструмент
 **Синтаксис**

 ManifestToCode /manifest:\<Image Manifest file\<> \</language: Code Language> Optional Args>

 **Аргументы**

||||
|-|-|-|

|/imageIdClass|Имя imageIdClass и связанный с ним файл, созданный инструментом. Для опции языка СЗ генерируются только файлы .h.<br /><br /> По \<умолчанию: Manifest Путь>»MyImageIds. \<Лэнг Экст>|Необязательный|
|/monikerClass|Имя monikerClass и связанный файл, созданный инструментом. Для опции языка СЗ генерируются только файлы .h. Это игнорируется для языка VSCT.<br /><br /> По \<умолчанию: Манифест Путь>»MyMonikers. \<Лэнг Экст>|Необязательный|

 **Примеры**

- Манифесттокод /манифест:D:'MyManifest.imagemanifest /язык:CSharp

- Манифесттокод /манифест:D:'MyManifest.imagemanifest /язык:C ' /namespace:My::Namespace /imageIdClass:MyImageIds /monikerClass:MyMonikers /classAccess:friend

- ManifestToCode /манифест:D:'MyManifest.imagemanifest /язык:VSCT /imageIdClass:MyImageIds

## <a name="notes"></a>Примечания

- Мы рекомендуем использовать этот инструмент с манифестами изображений, которые были сгенерированы манифестом из инструмента Ресурсов.

- Инструмент смотрит только на записи символов для создания оберток кода. Если манифест изображения не содержит символов, то генерируемые обертки кода будут пустыми. Если в манифесте изображения есть изображение или набор изображений, которые не используют символы, то они будут исключены из кодовой обертки.
