# <a name="test-area-5-change-source-control"></a>Область тестирования 5. Изменение системы управления версиями
Эта тестовая область подключаемого ключа охватывает изменение управления исходным элементом с помощью команды **управления исходным управлением change.**

 Команда **управления исходным элементом изменения** предоставляет четыре основные функции для пользователя:

- **Привязки:**

   Позволяет пользователю установить или восстановить источник управления между решением/проектом и магазином версий.

- **Развязать:**

   Удаляет проект/решение из элемента управления исходным элементом на основе подключения.

- **Подключение/отключение:**

  Переключения подключены или в автономном режиме состояние контролируемого решения, которое покрыто в зоне 3. Для получения дополнительной информации [см.](../../extensibility/internals/test-area-3-check-out-undo-checkout.md)

## <a name="command-menu-access"></a>Доступ к меню командования
 В [!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)] тестовых случаях используется следующий интегрированный путь меню среды разработки.

 Контроль источника изменений:**Файл**, **Управление исходом**, **Управление исходом изменения**.

## <a name="test-cases"></a>Тестовые случаи

### <a name="case-5b-unbind"></a>Дело 5b: Unbind
 Unbind удаляет информацию о контроле исходного кода из проектов и их решения. Затронутые проекты и решения основаны на сочетании выбора пользователя и того, как элементы были добавлены в элементы управления исходным элементом.

|Действие|Тестовые шаги|Ожидаемые результаты для проверки|
|------------|----------------|--------------------------------|
|Решение Unbind, содержащее одну файловую систему или локальный веб-проект IIS и один клиентский проект|1. Создайте файловую систему или локальный веб-проект IIS.<br />2. Добавьте решение в элемент управления исходом.<br />3. Добавьте новый клиентский проект в решение.<br />4. Примите проверить из решения, если предложено.<br />5. Откройте диалоговую коробку **управления исходом изменений.**<br />6. Нажмите **Unbind**.<br />7. Нажмите **OK,** чтобы закрыть диалоговую коробку.<br />8. Попытка проверить решение, проект, элементы решения, элементы проекта.|Решение и проекты НЕ находятся под контролем источников.<br /><br /> Команды меню управления исходным элементом не отображаются.|
