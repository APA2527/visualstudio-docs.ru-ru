# <a name="what39s-new-in-the-source-control-plug-in-api-version-12"></a>Что&#39;нового в подключаемый API версии 1.2
В этом разделе освещается расширенная функция управления исходным кодом в версии API 1.2. Ниже перечислены новые функции в версии 1.2:

## <a name="changes"></a>Изменения
- [Ликвидация файлов SAK](../../extensibility/internals/elimination-of-tilde-sak-files.md) Объясняет, как API-файл управления исходным элементом взаимодействует с временными файлами.

- [Применение параметров по нескольким соединениям проекта](../../extensibility/internals/application-of-settings-across-multiple-project-connections.md) Иллюстрирует, как плагин управления исходным кодом может использовать пакетную операцию для выполнения одной и той же операции управления исходным кодом в нескольких проектах или нескольких контекстах подключения.

- [Создание родительских контейнерных папок для решений](../../extensibility/internals/creating-parent-container-folders-for-solutions.md) Объясняет, как подключаемый API версия API 1.2 позволяет пользователю указать единое место управления корнем для всех веб-проектов в решении.

- [Дополнительное сравнение локального проекта Folder для магазина управления исходным управлением](../../extensibility/internals/optional-comparison-of-local-project-folder-to-source-control-store.md) Объясняет, как API-файл управления исходным элементом сравнивает версии файлов.

- [Удаление информации о контроле источника из . Proj и . Sln Files](../../extensibility/internals/removal-of-source-control-information-from-dot-proj-and-dot-sln-files.md) объясняет, как подключаемый API Версия 1.2 сохраняет информацию о проектах и решениях.
