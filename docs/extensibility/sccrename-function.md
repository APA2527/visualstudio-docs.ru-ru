# <a name="sccrename-function"></a>Функция SccRename
Эта функция переименовывает файл в систему управления исходным источником.

## <a name="syntax"></a>Синтаксис

```cpp
SCCRTN SccRename(
   LPVOID pvContext,
   HWND   hWnd,
   LPCSTR lpFileName,
   LPCSTR lpNewName
);
```

#### <a name="parameters"></a>Параметры
 pvКонтекст

(в) Структура контекста управления исходным элементом.

 Hwnd

(в) Ручка к окну IDE, которую плагин управления исходным элементом может использоваться в качестве родительского элемента для любых диалоговых коробок, которые она предоставляет.

 lpFileName

(в) Полностью квалифицированное имя файла, переименованного.

 lpNewName

[in] Новое полное имя. Если путь каталога отличается, файл перемещается из одного субдиректора в другой.

## <a name="return-value"></a>Возвращаемое значение
 Ожидается, что внедрение этой функции элемента управления исходным элементом вернет одно из следующих значений:

|Значение|Описание|
|-----------|-----------------|
|SCC_OK|Операция по переименованию успешно завершена.|
|SCC_E_PROJNOTOPEN|Проект не находится открытым под контролем исходного кода.|
|SCC_E_FILENOTCONTROLLED|Файл не находится в системе управления версиями.|
|SCC_E_ACCESSFAILURE|Возникла проблема с доступом к системе управления исходным кодом, вероятно, из-за проблем с сетью или раздором.|
|SCC_E_NOTAUTHORIZED|Пользователь не уполномочен выполнять эту операцию.|
|SCC_E_COULDNOTCREATEPROJECT|Проект не может быть создан в рамках процесса переименования.|
|SCC_E_OPNOTPERFORMED|Операция не была выполнена.|
|SCC_E_NONSPECIFICERROR|Произошла неопределенная или общая ошибка.|

## <a name="remarks"></a>Примечания
 Эта функция может использоваться для переименования файла или перемещения его из одного места в другое в системе управления исходным источником. Плагин управления исходным кодом не должен пытаться получить доступ к файлу на диске. Ответственность за переименование локального файла лежит на IDE.
