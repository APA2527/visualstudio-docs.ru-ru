# <a name="locationfieldmruprefix-element-visual-studio-templates"></a>Элемент LocationFieldMRUPrefix (Шаблоны визуальной студии)
Оначисление наиболее недавно используемых (MRU) путей в **новом проекте** и **добавление** нового элемента диалогового окна.

## <a name="syntax"></a>Синтаксис

```xml
<LocationFieldMRUPrefix> ... </LocationFieldMRUPrefix>
```

## <a name="attributes-and-elements"></a>Элементы и атрибуты
 В следующих разделах описаны атрибуты, дочерние и родительские элементы.

### <a name="attributes"></a>Атрибуты
 Нет.

### <a name="child-elements"></a>Дочерние элементы
 Нет.

### <a name="parent-elements"></a>Родительские элементы

|Элемент|Описание|
|-------------|-----------------|
|[TemplateData](../extensibility/templatedata-element-visual-studio-templates.md)|Определяет категорию шаблона и то, отображается ли он в диалоговом окне **Новый проект** или **Добавить новый элемент** .|

## <a name="remarks"></a>Примечания
 Этот элемент должен использоваться только для [!INCLUDE[vsipprvsip](../extensibility/includes/vsipprvsip_md.md)]шаблонов, производимых через .
