# <a name="sccdiff-function"></a>Функция SccDiff
Эта функция отображает (или по желанию просто проверяет) различия между текущим файлом (на локальном диске) и его последней зарегистрированной версией в системе управления исходным источником.

## <a name="syntax"></a>Синтаксис

```cpp
SCCRTN SccDiff(
   LPVOID    pvContext,
   HWND      hWnd,
   LPCSTR    lpFileName,
   LONG      fOptions,
   LPCMDOPTS pvOptions
);
```

### <a name="parameters"></a>Параметры
 pvКонтекст

(в) Структура контекста управления исходным элементом.

 Hwnd

(в) Ручка к окну IDE, которую плагин управления исходным элементом может использоваться в качестве родительского элемента для любых диалоговых коробок, которые она предоставляет.

 lpFileName

(в) Имя файла, для которого запрашивается разница.

 fВарианты

(в) Флаги командования. Дополнительные сведения см. в разделе "Примечания".

 pvOptions

(в) Опции управления исходным управлением.

## <a name="return-value"></a>Возвращаемое значение
 Ожидается, что внедрение этой функции элемента управления исходным элементом вернет одно из следующих значений:

|Значение|Описание|
|-----------|-----------------|
|SCC_OK|Рабочая копия и серверная версия идентичны.|
|SCC_I_FILESDIFFERS|Рабочая копия отличается от версии под контролем источника.|
|SCC_I_RELOADFILE|Файл или проект необходимо перезагрузить.|
|SCC_E_FILENOTCONTROLLED|Файл не находится в системе управления версиями.|
|SCC_E_NOTAUTHORIZED|Пользователю не разрешается выполнять эту операцию.|
|SCC_E_ACCESSFAILURE|Возникла проблема с доступом к системе управления исходным кодом, вероятно, из-за проблем с сетью или раздором. Рекомендуется повторная попытка.|
|SCC_E_NONSPECIFICERROR|Неспецифический сбой; разница в файлах не была получена.|
|SCC_E_FILENOTEXIST|Локальный файл не найден.|

## <a name="remarks"></a>Примечания
 Эта функция служит двум различным целям. По умолчанию отображается список изменений в файле. Плагин управления исходным элементом открывает свое собственное окно, в любом формате он выбирает, чтобы отобразить различия между файлом пользователя на диске и последней версией файла под контролем источника.

 Кроме того, IDE может просто определить, изменялся ли файл. Например, IDE может потребоваться определить, безопасно ли проверять файл без уведомления пользователя. В этом случае IDE проходит `SCC_DIFF_CONTENTS` в флаге. Плагин управления исходным элементом должен проверить файл на диске, байт байтом, против файла, управляемого исходным элементом, и вернуть значение, указывающее, отличаются ли эти два файла, не отображая ничего пользователю.

 В качестве оптимизации производительности плагин управления исходным элементом может использовать альтернативу, основанную на чексуме `SCC_DIFF_CONTENTS`или метке времени, вместо сравнения байт-байт, к которым призывает: эти формы сравнения, очевидно, быстрее, но менее надежны. Не все системы управления исходным управлением могут поддерживать эти альтернативные методы сравнения, и плагин, возможно, придется вернуться к сравнению содержимого. Все плагины управления исходным элементом должны, как минимум, поддерживать сравнение содержимого.

> [!NOTE]
> Быстрые флаги разницы являются взаимоисключающими. Это действительно, чтобы пройти не флаги, но это не действительно, чтобы одновременно пройти более одного. `SCC_DIFF_QUICK_DIFF`, которая представляет собой маску, которая сочетает в себе все флаги, может быть использован для тестирования, но она никогда не должна быть принята в качестве параметра.

|`fOption`|Значение|
|---------------|-------------|
|SCC_DIFF_IGNORECASE|Сравнение без чувственного случая (может быть использовано для быстрого или визуального различия).|
|SCC_DIFF_IGNORESPACE|Игнорирует белое пространство (может быть использовано для быстрого или визуального различия).|
|SCC_DIFF_QD_CONTENTS|Silently сравнивает файл, байт байтом.|
|SCC_DIFF_QD_CHECKSUM|Silently сравнивает файл через чексум при поддержке. Если не поддерживается, падает обратно к сравнению содержимого.|
|SCC_DIFF_QD_TIME|Silently сравнивает файл через его отметку времени при поддержке. Если не поддерживается, падает обратно к сравнению содержимого.|
