# <a name="extending-user-settings-and-options"></a>Расширение параметров и настроек пользователя
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

В Visual Studio есть два основных способа сохранения пользовательских настроек. Страницы « **Сервис/параметры** » позволяют пользователям задавать собственные значения для различных типов параметров, как общих, так и зависящих от языка. Параметры пользователя позволяют пользователям указывать общие свойства конфигурации Visual Studio. Дополнительные сведения о страницах параметров см. в разделе [Параметры и страницы](../extensibility/internals/options-and-options-pages.md)параметров. Дополнительные сведения о параметрах пользователей см. в разделе [Поддержка параметров пользователя](../extensibility/internals/support-for-user-settings.md).
