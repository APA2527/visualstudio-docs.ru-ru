# <a name="enabling-the-azure-sdk"></a>Включение пакета Azure SDK
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Пакет Azure SDK для .NET позволяет создавать, настраивать, создавать, отлаживать, выполнять и упаковывать масштабируемые веб-приложения и службы в Azure.  
  
 Пакет SDK для Azure можно скачать из диалогового окна **Новый проект** .  
  
 [!INCLUDE[note_settings_general](../includes/note-settings-general-md.md)]  
  
### <a name="to-enable-the-azure-sdk"></a>Включение пакета SDK для Azure  
  
1. В меню **файл** создайте новый проект. (Нажмите кнопку **создать** , а затем выберите **проект**.)  
  
2. Разверните узел для **Visual Basic** или **C#** , а затем щелкните **облако**.  
  
3. В средней области выберите **получить пакет Azure SDK для .NET**и нажмите кнопку **ОК**.  
  
     Следуйте инструкциям установщика.  
  
    > [!NOTE]
    > Для установки средств Azure необходимо установить службы IIS 7,0 и компоненты разработки приложений ASP.NET. Если эти компоненты не установлены, программа установки средств Azure предложит вам установить их.
