# <a name="asyncvoidmethodbuilderm_objectidfordebugger-field"></a>Поле AsyncVoidMethodBuilder.m_objectIdForDebugger
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Представляет объект с отложенной инициализацией, используемый отладчиком для уникальной идентификации этого построителя.  
  
 **Пространство имен:** <xref:System.Runtime.CompilerServices?displayProperty=fullName>  
  
 **Сборка:** mscorlib (в mscorlib.dll)  
  
 Так как вы не можете получить доступ к этому внутреннему элементу из .NET Framework, на стандартном промежуточном языке (CIL) приведен следующий синтаксис.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
.field private object m_objectIdForDebugger  
```
