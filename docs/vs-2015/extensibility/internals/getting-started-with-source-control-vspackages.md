# <a name="getting-started-with-source-control-vspackages"></a>Начало работы с пакетом VSPackage системы управления версиями
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Пакет VSPackage системы управления версиями является альтернативой более сложных для подключаемого модуля системы управления версиями. (См. в разделе [Создание подключаемого модуля управления источника](../../extensibility/internals/creating-a-source-control-plug-in.md) узнать больше о источника управления подключаемых модулей.) Пакет VSPackage системы управления версиями обеспечивает полный контроль над модель управления исходным кодом, функций и пользовательский интерфейс (UI), а также интегрируется в [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] среде, что VSPackage.
