# <a name="could-not-retrieve-schema-information-for-database-object-ltobject-namegt"></a>Не удалось получить сведения о схеме для объекта базы данных &lt;object имя &gt;
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Это сообщение обычно появляется, когда объект в **Обозреватель сервера или Обозреватель базы данных** копируется в буфер обмена, удаляется из базы данных и затем добавляется в конструктор. Поскольку объект базы данных больше не существует, появляется это сообщение.

### <a name="to-correct-this-error"></a>Исправление ошибки

- Нажмите кнопку **OK**, чтобы закрыть диалоговое окно.
