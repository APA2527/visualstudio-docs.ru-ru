# <a name="c28227"></a>C28227
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28227: Аннотация не может также быть PrimOp (см. предыдущее объявление)  
  
 Это предупреждение означает, что именованной аннотации объявляется как обычные заметки и также была ранее объявлена как PrimOp. Это не должно быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не ожидается возникают в примерах.
