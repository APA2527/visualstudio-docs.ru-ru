# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28213: заметка `_Use_decl_annotations_` должна использоваться для ссылки, без изменения, перед объявлением. \<Зачем >  
  
 Это предупреждение выводится при указании ссылки на более раннее объявление без обязательного `_Use_decl_annotations_` аннотации.
