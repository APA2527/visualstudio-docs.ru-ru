# <a name="c28283"></a>C28283
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

предупреждение C28283: для Symbol Указанная спецификация размера пока не поддерживается.  
  
 Предупреждение означает, что заметка "Sentinel" в спецификации размера получила значение, отличное от нуля. По сути, вызывающая сторона пыталась сказать, что строка завершается символом, отличным от двоичного нуля.
