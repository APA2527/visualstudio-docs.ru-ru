# <a name="c1252"></a>C1252
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

предупреждение C1252: циклическая или отсутствующая зависимость между подключаемыми модулями: требуется GUID  
  
 Средство анализа кода сообщает об этом предупреждении при внутренней ошибке с зависимостями подключаемого модуля, а не в анализируемом коде.
