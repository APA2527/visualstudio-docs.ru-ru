# <a name="c1250"></a>C1250
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

предупреждение C1250: не удалось загрузить подключаемый модуль.  
  
 Средство анализа кода сообщает об этом предупреждении при возникновении внутренней ошибки в подключаемом модуле, а не в анализируемом коде.
