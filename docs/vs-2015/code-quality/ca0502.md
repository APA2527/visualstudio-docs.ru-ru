# <a name="ca0502"></a>CA0502

Неподдерживаемый язык.

Эта ошибка возникает, когда пользователь пытается проанализировать проект ASP.NET, содержащий одну или несколько страниц или классов, написанных на языке, C# отличном от или Visual Basic. Чтобы устранить эту ошибку, удалите файл из проекта или отключите анализ кода.
