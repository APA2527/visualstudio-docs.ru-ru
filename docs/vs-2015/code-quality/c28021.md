# <a name="c28021"></a>C28021
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28021: Параметр \<param >, аннотированный с использованием \<anno > должен быть указателем  
  
 Это предупреждение возникает в том случае, если объект, аннотированный не является типом указателя. Эта заметка не может использоваться с `void` или целочисленных типов.
