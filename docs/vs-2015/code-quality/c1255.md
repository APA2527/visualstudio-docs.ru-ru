# <a name="c1255"></a>C1255
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

предупреждение C1255: Данные PCH для подключаемого модуля имеют неправильную длину.  
  
 Средство анализа кода сообщает об этом предупреждении при возникновении внутренней ошибки в средстве, а не в анализируемом коде.
