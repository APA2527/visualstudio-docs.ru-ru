# <a name="c28204"></a>C28204
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28204: \<Function >: только одна из этих перегрузок и \<с именем файла > (\<Line >) записываются для \<paramName >: не следует добавлять заметки к обоим параметрам или ни одному из них.  
  
 Это предупреждение появляется при возникновении ошибки в заметках.
