# <a name="c28290"></a>C28290
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28290: аннотация для функции содержит превышает фактическое число параметров  
  
 Средство анализа кода сообщает это предупреждение, когда аннотация для функции содержит превышает фактическое число параметров.
