# <a name="ca0063"></a>CA0063

Не удалось загрузить файл набора правил или один из его зависимых файлов набора правил.

Не удалось загрузить файл набора правил или один из его зависимых файлов набора правил.

Не удалось найти указанный набор правил или не удалось найти один из файлов набора правил, включенных в набор правил. Убедитесь, что все наборы правил, включенные в набор правил, существуют на диске и что в проекте указаны соответствующие каталоги набора правил с помощью свойства **кодеаналисисрулесетдиректориес** в MSBuild.

Чтобы выполнить отладку ошибки, проверьте RuleSet-файл в текстовом редакторе. Чтобы найти путь к RuleSet-файлу, щелкните правой кнопкой мыши проект в обозреватель решений, выберите пункт **Свойства**, а затем щелкните **анализ кода**. Убедитесь, что в поле **выполнить этот набор правил**выбран файл RuleSet. Путь к набору правил указан в поле Описание.

Проверьте значение атрибута **path** для всех элементов **include** . Пути включения могут использовать относительные пути к родительскому или текущему RuleSet-файлу, переменным среды и абсолютным путям. Пример:

```
<Include Path="%PUBLIC%\Documents\RuleSets\alldesign.ruleset" Action="Default" />
<Include Path="..\alldesign.ruleset" Action="Default" />
<Include Path="C:\Rulesets\alldesign.ruleset" Action="Default" />
```

Проверьте каждый из этих включаемых путей и убедитесь, что они действительны.

В некоторых случаях расположение набора правил может зависеть от свойства MSBuild. На свойства MSBuild нельзя ссылаться из набора правил. Чтобы обойти эту ошибку, укажите дополнительные пути поиска в свойстве **Кодеаналисисрулесетдиректориес** MSBuild. В этом сценарии укажите только имя набора правил в атрибуте **path** элемента **include** .
