# <a name="c28212"></a>C28212
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28212: Для аннотации ожидалось выражение указателя  
  
 Это предупреждение указывает параметр нумерованной в заметку (не функцию, аннотированный) должен быть указатель или тип массива, что была обнаружена другого типа. Заметку необходимо исправить.
