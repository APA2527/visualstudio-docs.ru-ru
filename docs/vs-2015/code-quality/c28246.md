# <a name="c28246"></a>C28246
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28246: Аннотация для функции " \<name> "-параметр " \<parameter> " не соответствует типу параметра  
  
 Оператор __deref был применен к типу, не являющемуся указателем, при создании заметки.
