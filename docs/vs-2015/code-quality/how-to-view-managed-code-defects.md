# <a name="how-to-view-managed-code-defects"></a>Практическое руководство. Просмотр дефектов управляемого кода
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

При анализе кода ошибки, предупреждения и сообщения выводятся в окне "анализ кода". Окно Анализ кода отображается автоматически при выполнении анализа кода для проекта или решения.

 В окне "анализ кода" можно выполнять следующие задачи:

- **Выберите заголовок предупреждения.**

     Дополнительные сведения о предупреждении отображаются в окне "анализ кода". Строка исходного файла, которая вызвала предупреждение, отображается и выделяется в редакторе исходного кода.

- **Выберите идентификатор предупреждения.** Идентификаторы предупреждений имеют префикс **CA** и включают четырехзначное число. Например, CA1234.

     Отобразится раздел справки по библиотеке MSDN для предупреждения. Раздел справки содержит подробные сведения о причине предупреждения и содержит примеры, демонстрирующие причины возникновения предупреждения и разрешения.
