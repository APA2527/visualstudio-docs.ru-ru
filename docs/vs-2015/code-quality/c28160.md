# <a name="c28160"></a>C28160
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

предупреждение C28160: Аннотация ошибки  
  
 Это предупреждение появляется при `__drv_error` обнаружении заметки. Эта заметка используется для определения методов программирования, которые следует исправить, и может использоваться с `__drv_when` заметкой для указания определенных сочетаний параметров.
