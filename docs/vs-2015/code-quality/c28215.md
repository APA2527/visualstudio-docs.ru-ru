# <a name="c28215"></a>C28215
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28215: typefix нельзя применить к параметру, который уже имеет typefix  
  
 Применение `__typefix` заметки к параметру, который уже имеет эту заметку, является ошибкой. `__typefix`Заметки используются только в нескольких особых случаях, и это предупреждение не ожидается в обычном режиме использования.
