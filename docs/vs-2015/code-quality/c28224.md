# <a name="c28224"></a>C28224
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

предупреждение C28224: для аннотации требуются параметры  
  
 Это предупреждение означает, что Именованная заметка используется без параметров и требуется по крайней мере один параметр. Это не должно быть возможным, если стандартные макросы используются для заметок; Это предупреждение не ожидается при обычном использовании.
