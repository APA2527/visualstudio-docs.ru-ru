# <a name="c28224"></a>C28224
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28224: Аннотации требуются параметры  
  
 Это предупреждение указывает, что именованной аннотации используется без параметров, и по крайней мере один параметр является обязательным. Это не должно быть возможно, если стандартные макросы, используемые для заметок; Это предупреждение не должен рассматриваться при нормальном использовании.
