# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>Оценка сложности и удобства сопровождения управляемого кода
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Повышенная сложность современных программных приложений также увеличивает усложняет код, надежный и удобный в сопровождении. В последние годы многие меры программного обеспечения, известных как метрики кода, были разработаны, может помочь разработчикам понять, где их код должен переработки или усиленного тестирования.  
  
 Разработчики могут использовать [!INCLUDE[vsprvsts](../includes/vsprvsts-md.md)] для получения данных метрик кода, которые измеряют сложности и удобства сопровождения управляемого кода. Данные метрик кода могут создаваться для всего решения или отдельного проекта.  
  
 В этом разделе описывается, как создать и использовать показатели качества кода в [!INCLUDE[vsprvs](../includes/vsprvs-md.md)].
