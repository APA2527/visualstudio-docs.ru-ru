# <a name="c28230"></a>C28230
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28230: тип параметра не имеет члена.  
  
 Это предупреждение означает, что аргумент в заметке пытается получить доступ к `struct`, `class`или `union`, а именованный элемент не существует.
