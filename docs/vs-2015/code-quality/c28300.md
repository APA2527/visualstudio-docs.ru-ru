# <a name="c28300"></a>C28300
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C28300: < имя_параметра >: Выражение несовместимые типы операндов для оператора < имя_оператора >  
  
 Это предупреждение возникает в том случае, когда типы операндов в параметре не совместимы с оператором.
