# <a name="c6995"></a>C6995
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Предупреждение C6995: Не удалось сохранить файл XML-журнала  
  
 Это предупреждение означает, что средство анализа кода не удается создать журнал дефект, который является результатом выполнения анализа кода.  
  
 Эта ошибка может указывать на сбой диска или указать, что у вас нет разрешения на создание файла в указанном каталоге.
