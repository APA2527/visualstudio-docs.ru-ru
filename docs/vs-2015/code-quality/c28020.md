# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28020: в \<expr> этом вызове выражение не равно true  
  
 Это предупреждение выводится, если \_ \_ выражение удовлетворяет указанному выражению. Часто это указывает на неверный параметр.  
  
 Если это происходит в объявлении функции, заметки указывают на невозможность условия.
