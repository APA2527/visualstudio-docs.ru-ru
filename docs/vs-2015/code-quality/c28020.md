# <a name="c28020"></a>C28020
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28020: выражение \<expr > не равно true в этом вызове  
  
 Это предупреждение выводится, если \_удовлетворяет\_ выражению, которое не имеет значения true. Часто это указывает на неверный параметр.  
  
 Если это происходит в объявлении функции, заметки указывают на невозможность условия.
