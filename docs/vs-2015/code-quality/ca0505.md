# <a name="ca0505"></a>CA0505

Нерекомендуемое свойство будет пропущено, так как свойство суперцединг определено.

Предупреждение CA505 создается, если как заданные свойства нерекомендуемых, так и суперцедингного анализа кода содержат значения в проекте или файле конфигурации сборки. Чтобы устранить эту проблему, удалите устаревшее свойство.
