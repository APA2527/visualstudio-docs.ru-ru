# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Warning C28280: для Symbol обнаружен "End" без соответствующего "Begin  
  
 Язык аннотации поддерживает конструкцию Begin и End (`{` и `}` в C), а связывание не сбалансировано. Эту ситуацию можно избежать, если используются макросы.
