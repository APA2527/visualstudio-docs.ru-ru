# <a name="c28221"></a>C28221
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

предупреждение C28221: для параметра в аннотации ожидалось строковое выражение  
  
 Это предупреждение означает, что параметр в заметке должен быть строкой, а обнаружен другой тип. Обычно это указывает на неправильное написание макроса аннотации и не ожидается в обычном использовании.
