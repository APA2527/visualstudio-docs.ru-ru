# <a name="help-menu-commands"></a>Команды меню "Справка"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

В меню **Справка** содержатся команды для открытия файлов справки.  
  
 **Index**  
 Открывает экран содержимое для справки по Spy + +.  
  
 **Использование справки**  
 Открывает файл справки, в котором описывается использование справки.  
  
 **О программе Spy++**  
 Показывает версию программного обеспечения и сведения об авторских правах для Spy + +.
