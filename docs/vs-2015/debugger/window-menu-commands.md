# <a name="window-menu-commands"></a>Команды меню "Окно"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

В меню **Окно** доступны команды для управления окнами Spy++.  
  
 **Аргумент**  
 Переупорядочивает все окна в шаблоне Cascade.  
  
 **Сверху вниз**  
 Переупорядочение всех окон в шаблоне горизонтальной плитки.  
  
 **Слева направо**  
 Переупорядочивает все окна в шаблоне вертикальной плитки.  
  
 **Упорядочить значки**  
 Переупорядочивает значки представлений, если окна были сведены к сведению.  
  
 **Закрыть или закрыть все**  
 Закрывает одно или все окна в Spy + +.  
  
 **Обновить**  
 Обновляет выбранное в данный момент представление.  
  
 **Окно**  
 Позволяет выбрать конкретное окно.
