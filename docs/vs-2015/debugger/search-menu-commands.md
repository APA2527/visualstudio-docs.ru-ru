# <a name="search-menu-commands"></a>Команды меню "Поиск"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

В меню **Поиск** содержатся команды для поиска конкретных окон.  
  
 **Поиск окна, процесса, потока или сообщения**  
 Открывает **Window** / **Process** / **Thread** / диалоговое окно**мессажесеарч** потока процесса, в котором можно использовать **финдертул** или указать критерии поиска.  
  
 **Следующий**  
 Выполняет поиск следующего соответствующего окна, процесса, потока или сообщения в текущем представлении. Этот пункт меню (и соответствующая кнопка на панели инструментов) доступен только при наличии допустимого результата поиска, который не является уникальным.  
  
 **Найти ранее**  
 Выполняет поиск предыдущего соответствующего окна, процесса, потока или сообщения в текущем представлении. Этот пункт меню (и соответствующая кнопка на панели инструментов) доступен только при наличии допустимого результата поиска, который не является уникальным.
