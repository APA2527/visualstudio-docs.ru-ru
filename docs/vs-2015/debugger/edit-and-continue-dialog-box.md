# <a name="edit-and-continue-dialog-box"></a>Изменить и продолжить - диалоговое окно
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Только машинный код). это диалоговое окно появляется при редактировании кода во время отладки и **изменения и продолжения** распознавания того, что он может применить изменения кода. Дополнительные сведения см. в разделе [Изменить и продолжить](../debugger/edit-and-continue.md).  
  
 **Больше не показывать это окно**  
 Если установить этот флажок, то **изменить и продолжить** будут применять изменения кода без запроса разрешения в будущем. Данное предупреждение можно включить снова, если зайти в диалоговое окно **Параметры**, открыть папку **Отладка**, выбрать страницу **Изменить и продолжить**, а затем выбрать **Вызывать командами отладки** и **Запрос**.
