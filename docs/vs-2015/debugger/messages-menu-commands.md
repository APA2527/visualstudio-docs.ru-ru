# <a name="messages-menu-commands"></a>Команды меню "Сообщения"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

В меню **Сообщения** содержатся команды для управления представлениями сообщений.  
  
 **Ведение журнала запуска и отмены**  
 Запускает или останавливает представление сообщений.  
  
 **Параметры ведения журнала**  
 Открывает диалоговое окно **мессажеоптионс** , в котором можно указать условия поиска сообщений.  
  
 **Очистить журнал**  
 Очищает содержимое представления активных сообщений.
