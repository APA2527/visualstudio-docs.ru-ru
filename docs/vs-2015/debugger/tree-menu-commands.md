# <a name="tree-menu-commands"></a>Команды меню "Дерево"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Меню **Дерево** содержит команды для развертывания или свертывания представлений.  
  
 **Развернуть на один уровень**  
 Раскрывает выбранный в данный момент элемент в системном представлении, отображая один уровень под ним.  
  
 **Развернуть ветвь**  
 Разворачивает все уровни под выбранным в данный момент элементом.  
  
 **Развернуть все**  
 Развертывает и отображает все уровни в системном представлении.  
  
 **Свернуть**  
 Сворачивает все уровни под выбранным элементом.
