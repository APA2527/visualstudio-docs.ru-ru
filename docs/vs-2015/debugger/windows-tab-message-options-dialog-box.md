# <a name="windows-tab-message-options-dialog-box"></a>Вкладка "Окна" диалогового окна "Параметры сообщения"
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

С помощью вкладки **Окна** можно задать область сообщений, указанных в [представлении сообщений](../debugger/messages-view.md). Чтобы открыть диалоговое окно [Параметры сообщений](../debugger/message-options-dialog-box.md), выберите пункт **Записывать сообщения** в меню **Spy**.  
  
 На вкладке **Окна** доступны следующие параметры:  
  
 **Инструмент поиска**  
 Средство перетаскивания, используемое для выбора окна.  
  
 **Скрыть Spy + +**  
 Удаляет главное окно Spy + + и оставляет диалоговое окно **мессажеоптионс** видимым поверх других приложений.  
  
 **Родительское окно**  
 Отображение сообщений для выбранного окна и его непосредственного родительского окна.  
  
 **Дочернее окно**  
 Отображение сообщений для выбранного окна и всех его дочерних окон, включая вложенные дочерние окна.  
  
 **Окна одного потока**  
 Отображение сообщений для выбранного окна и всех других окон, принадлежащих одному потоку.  
  
 **Окна одного процесса**  
 Отображение сообщений для выбранного окна и всех других окон, принадлежащих одному процессу.  
  
 **Все окна в системе**  
 Отображение сообщений для всех окон.  
  
 **Сохранить в качестве параметров по умолчанию**  
 Сохраните указанные выше параметры для новых окон потока сообщений. Также параметры автоматически сохраняются при выходе из Spy++.
