# <a name="lib-task"></a>Задача LIB
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Заключает в оболочку 32-разрядный диспетчер библиотек Майкрософт lib.exe. Диспетчер библиотек создает библиотеку объектных файлов в формате COFF и управляет ею. Диспетчер библиотек также может создавать файлы экспорта и импортировать библиотеки в экспортированные справочные определения. Дополнительные сведения см. в разделе [Справочник по lib](https://msdn.microsoft.com/library/ecc7f643-bbd4-47a3-8dc6-b360f880db91) и [Запуск lib](https://msdn.microsoft.com/library/d54f5c81-7147-4b2c-a8db-68ce6eb1eabd).
