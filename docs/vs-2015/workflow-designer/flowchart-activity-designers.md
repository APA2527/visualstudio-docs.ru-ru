# <a name="flowchart-activity-designers"></a>Конструктор действия блок-схемы
[!INCLUDE[wfd1](../includes/wfd1-md.md)] включает определенное количество системных действий, которые могут быть использованы для построения рабочих процессов. Данный раздел содержит действия, предоставленные системой и используемые для управления блок-схемами в пределах рабочего процесса. В следующих подразделах приводится описание этих действий, а также описание их использования.

## <a name="in-this-section"></a>Содержание
 [Блок-схема](../workflow-designer/flowchart-activity-designer.md) **, используемая для создания рабочих процессов, которые определяют сложные элементы управления потоком и управляют ими.**  [FlowDecision](../workflow-designer/flowdecision-activity-designer.md) Предоставляет ветвь для потока управления в один из двух вариантов в зависимости от того, удовлетворяет ли заданное условие.

 [FlowSwitch \<T >](../workflow-designer/flowswitch-t-activity-designer.md) Предоставляет ветвление для потока управления на основе критерия соответствия, если требуются более двух альтернативных ветвей.
