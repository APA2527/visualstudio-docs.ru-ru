# <a name="accessibility-tips-and-tricks"></a>Специальные возможности. Советы и рекомендации
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

В [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] встроена поддержка специальных возможностей, что позволяет облегчить работу с клавиатурой, использовать программы чтения с экрана или другие вспомогательные технологические устройства. Раздел включает в себя ряд предложений по оптимизации [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] для специальных возможностей, а также некоторые полезные сочетания клавиш.

> [!NOTE]
> Отображаемые диалоговые окна и команды меню могут отличаться от описанных в справке в зависимости от текущих параметров или выпуска. Чтобы изменить параметры, выберите в меню **Сервис** пункт **Импорт и экспорт параметров** . Дополнительные сведения см. [в разделе Настройка параметров разработки в Visual Studio](https://msdn.microsoft.com/22c4debb-4e31-47a8-8f19-16f328d7dcd3).

## <a name="save-your-ide-settings"></a>Сохранение параметров интегрированной среды разработки
 Вы можете настроить интегрированную среду разработки на своем компьютере, сохранив макет окна, схему назначений клавиш и другие настройки. Дополнительные сведения см. [в разделе Настройка параметров разработки в Visual Studio](https://msdn.microsoft.com/22c4debb-4e31-47a8-8f19-16f328d7dcd3).

## <a name="common-windows-shortcut-key-combinations"></a>Стандартные сочетания клавиш Windows
 Для выполнения задач в [!INCLUDE[vsprvs](../../includes/vsprvs-md.md)] можно использовать многие стандартные сочетания клавиш Windows. Ниже приведены примеры нескольких сочетаний клавиш.

|Описание|Сочетание клавиш|
|-----------------|---------------------|
|Включение и выключение высокой контрастности|**ALT слева + SHIFT слева + Print Screen**|
|Установка или снятие флажка в диалоговом окне|**»**|
|Отображение свойств выбранного элемента. Например, перемещение фокуса в окно "Свойства", если выбрана форма, или открытие диалогового окна "Страницы свойств", если выбран проект.|**ALT + ВВОД**|
|Отображение элементов активного списка, например раскрывающегося списка, в диалоговом окне|**ALT**+СТРЕЛКА ВНИЗ|
|Отображение элементов в активном раскрывающемся списке в таблице|**SHIFT**  +  **ALT**  +  **стрелка вниз**|

## <a name="hidden-visual-studio-shortcut-key-combinations"></a>Скрытые сочетания клавиш Visual Studio
 Для некоторых функций доступны сочетания клавиш, не отображаемые на странице "Клавиатура" папки "Среда" диалогового окна "Параметры". Далее представлен список этих сочетаний клавиш.

|Компонент|Описание|Сочетание клавиш|
|-------------|-----------------|---------------------|
|Окно "Панель элементов"|Перемещение между вкладками панели элементов|**CTRL**  +  **Стрелка вверх**<br /><br /> и<br /><br /> **CTRL**  +  **Стрелка вниз**|
|Окно "Панель элементов"|Добавление элемента управления из панели элементов в форму или конструктор|**ВХОДИТЬ**|
|Страница "Клавиатура", папка "Среда", диалоговое окно "Параметры"|Удалить сочетание клавиш, заданное в параметре " **нажать сочетание** клавиш"|**BACKSPACE**|
|Все окна инструментов|Выбор первой кнопки на панели инструментов в окне|**SHIFT**  +  **ALT**|
|Панели инструментов IDE|Выбор первой кнопки на панели инструментов "Стандартная"|**ALT**, **CTRL** + **TAB** **Примечание.** Нажмите **CTRL** + **TAB** еще раз для выбора первой кнопки на следующей панели интерфейса IDE.|
