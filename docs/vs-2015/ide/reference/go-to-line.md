# <a name="go-to-line"></a>Переход к строке
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

Этого диалоговое окно позволяет перейти к определенной строке в активном документе. Чтобы перейти в это окно, откройте документ для редактирования, а затем выберите **Перейти** в меню **Правка**.

 Номер строки (1-`n`) позволяет ввести номер строки в активном документе, куда необходимо переместиться. Этот номер должен находиться в диапазоне от 1 до `n` (число строк в текущем документе).
