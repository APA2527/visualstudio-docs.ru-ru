# <a name="security-bibliography"></a>Библиография по вопросам безопасности
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Ниже представлены избранные электронные и печатные ресурсы, посвященные вопросам разработки безопасных приложений и настройки безопасных сред.

## <a name="microsoft-web-sites"></a>Веб-узлы корпорации Microsoft
 [Безопасность Майкрософт](https://www.microsoft.com/security) Узнайте, как работает корпорация Майкрософт, чтобы обеспечить безопасность приложений и систем, от настольной системы до систем уровня сети, со ссылками на ресурсы по безопасности для ИТ-специалистов, разработчиков и домашних пользователей.

 [Ресурсы по безопасности на сайте MSDN](https://www.microsoft.com/msrc?rtc=1) Руководство содержит документацию для разработчиков, примеры кода, технические статьи и другие ресурсы для разработки безопасных приложений.

 [Безопасность Microsoft TechNet](https://www.microsoft.com/securityengineering) Содержит указатели на наиболее важные технические сведения о безопасности для специалистов по информационным технологиям и опытных пользователей.
