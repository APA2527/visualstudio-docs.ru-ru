# <a name="wpf-targets-files"></a>TARGETS-файлы WPF

Windows Presentation Foundation (WPF) расширяет MSBuild, добавляя набор специальных задач WPF, которые будут объединены в специальный *TARGETS-файл*, *Microsoft.WinFX.targets*. Этот файл объединяет набор задач MSBuild, необходимых для построения проекта MSBuild в WPF.
