# <a name="projects-and-solutions-in-visual-studio-for-mac"></a>Проекты и решения в Visual Studio для Mac

В этой статье содержится обзор *проекта* и *решение* основные понятия в Visual Studio для Mac.

> [!NOTE] 
> Этот раздел относится к Visual Studio для Mac. Visual Studio в Windows см. в разделе [проектов и решений в Visual Studio](/visualstudio/ide/solutions-and-projects-in-visual-studio).

## <a name="projects"></a>Проекты

При создании нового приложения, веб-сайта, и т.д. в Visual Studio для Mac, сначала с проектом. Проект содержит все файлы, необходимые (исходный код, изображения, файлы данных, и т.д.), которые необходимы для компиляции исполняемого файла, библиотеки или веб-сайта.

Проект определяется с помощью файла (например, `.csproj` для C# проекты) который содержит файл xml, определяющий этот файл и иерархии папок, пути к файлам и параметры конкретного проекта, такие как параметры построения.

При загрузке проекта в Visual Studio для Mac, на панели решения использует файл проекта для отображения файлов и папок в проекте. Во время компиляции MSBuild считывает параметры из файла проекта для создания исполняемого файла.

## <a name="solutions"></a>Решения

Объект *решение* — это контейнер которой группируются вместе одного или нескольких связанных проектов. Описываются решения, в текстовый файл (расширение `.sln`) в собственном уникальном формате; он не предназначен для изменения вручную.

## <a name="managing-projects-in-the-solution-pad"></a>Управление проектами в панели решения

После проекта был создан или загружен, можно использовать для просмотра и управления, проекта или решения и файлы, содержащиеся в панели решения. На следующем рисунке показана панель решения с помощью решения .NET Core, которое содержит два проекта:

![Пример решения с несколькими проектами](media/solution-example.png)

Можно управлять свойствами проектов и решений можно дважды щелкнуть на имени проекта или решения, или, щелкнув правой кнопкой мыши и выбрав **параметры**.

Дополнительные сведения об этих параметрах см. в статье [Управление свойствами проекта и решения](managing-solutions-and-project-properties.md).
